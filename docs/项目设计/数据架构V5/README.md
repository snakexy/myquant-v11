## ⚡ 快速查询表

### 按业务类型查询

| 业务 | 交易时间 | 盘后时间 | 非交易时间 | 关键限制 |
|------|----------|----------|-----------|----------|
| L1快照 | TdxQuant(0.60ms) → XtQuant(6ms) | TdxQuant/XtQuant → PyTdx2 | PyTdx2(10-19ms) | - |
| 当天分钟线 | XtQuant(0.90ms) → PyTdx2 | XtQuant(9-15ms) → PyTdx2 | PyTdx2(10-19ms) | **TdxQuant不支持** |
| 历史K线 | QLib(7-10ms) → PyTdx2 | QLib/XtQuant(9-15ms) → PyTdx2 | QLib(7-10ms) → PyTdx2 | - |
| **板块数据** | **TdxQuant(6.99ms)** 🔥 | **PyTdx2(~15ms)** 🔥 真在线 | PyTdx2(~15ms) | **TdxQuant盘后只有昨天历史** |
| 财务数据 | TdxQuant → XtQuant → PyTdx2 | TdxQuant/XtQuant → PyTdx2 | PyTdx2 | - |
| L0订阅 | XtQuant(300股) → TdxQuant(100股) | 不推荐（市场休市） | 不支持 | - |

### 按数据源查询

| 数据源 | 最擅长 | 关键限制 | 可用时间 |
|--------|--------|----------|----------|
| **TdxQuant** | L1快照(0.60ms) + **板块(6.99ms, 586板块)** | **不支持当天分钟线** | 交易时间 |
| **XtQuant** | 当天分钟线(0.90ms) + 订阅(300股) | **不支持板块** + 读取本地缓存 | 交易+盘后 |
| **PyTdx2** | **24/7可用** | 响应稍慢 | **24/7** |
| **QLib** | 历史K线(7-10ms) | 不支持快照、板块 | **24/7** |

## 🎯 关键限制速查

⚠️ **开发前必读 - 触犯这些限制会导致功能失败**

| 数据源 | ❌ 关键限制 | 影响 |
|--------|-----------|------|
| **TdxQuant** | 不支持当天分钟线 | 当天分钟线必须用XtQuant/PyTdx2 |
| **TdxQuant** | 仅交易时间可用 | 非交易时间无法使用 |
| **TdxQuant** | 盘后板块数据只有昨天历史 | 盘后板块应用PyTdx2真在线 |
| **XtQuant** | **不支持板块数据（板块弱+代码映射不兼容）** | **板块业务完全无法使用** |
| **XtQuant** | 读取QMT本地缓存，非真正在线 | 获取当天数据需确保QMT已更新 |
| **XtQuant** | 仅交易+盘后可用 | 周末/节假日无法使用 |

## 📖 数据源核心定位

### TdxQuant - 交易时间快照和板块之王 🏆

| 能力 | 性能 | 地位 |
|------|------|------|
| L1 快照 | **0.60ms** | 交易时间最快 |
| 板块数据 | **6.99ms, 586板块** | 绝对优势（XtQuant不支持） |
| 历史K线 | ~50ms | 可用 |
| ❌ 当天分钟线 | - | **不支持** |

### XtQuant - 当天分钟线和订阅专家

| 能力 | 性能 | 地位 |
|------|------|------|
| 当天分钟线 | **0.90ms** | 交易时间最优 |
| 订阅功能 | **300股** | 可用 |
| ❌ 板块数据 | - | **不支持** |

### PyTdx2 - 非交易时间救星 🌟

| 能力 | 性能 | 地位 |
|------|------|------|
| **24/7可用性** | - | 非交易时间唯一选择 |
| 快照 | 10-19ms | 可用 |
| 当天分钟线 | 10-19ms | 可用 |
| 板块数据 | ~15ms | 非交易时间唯一 |

### QLib - 本地数据库

| 能力 | 性能 | 地位 |
|------|------|------|
| 历史K线 | **7-10ms** | 本地库最快 |

## 🔍 决策树

```
请求数据
│
├─ 需要板块数据？
│  ├─ 是 → 盘后/非交易时间？
│  │        ├─ 是 → PyTdx2 (~15ms) 🔥 真在线获取
│  │        └─ 否 → TdxQuant (6.99ms, 586板块) 🔥 盘中
│  │
│  └─ 否 → 需要当天分钟线？
│         ├─ 是 → 盘后/非交易时间？
│         │        ├─ 是 → QMT连接？
│         │        │        ├─ 是 → XtQuant(9-15ms) + PyTdx2(10-19ms) 🚀 并行分工
│         │        │        └─ 否 → PyTdx2 (10-19ms)
│         │        └─ 否 → XtQuant (0.90ms订阅缓存) 🔥 盘中最优
│  │
│  └─ 否 → 需要快照？
│         ├─ 是 → 盘后/非交易时间？
│         │        ├─ 是 → PyTdx2 (10-19ms)
│         │        └─ 否 → TdxQuant (0.60ms) 🔥 盘中最快
│         │
│         └─ 否 → 历史K线 → QLib (7-10ms) 🔥 本地最快
```

**💡 并行分工场景**：
- **触发条件**：盘后/非交易时间 + 需要当天分钟线 + QMT已连接 + 股票数>50只
- **分工方式**：XtQuant处理一半 + PyTdx2处理另一半，并行执行
- **性能提升**：整体吞吐量约2x
