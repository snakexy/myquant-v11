## 2. 按钮规范

### 2.1 基础按钮样式

```scss
// 使用原生<button> + .btn类
.btn {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: 8px;
  padding: 10px 20px;
  border: none;
  border-radius: 6px;
  font-size: 14px;
  font-weight: 500;
  cursor: pointer;
  transition: all 0.2s ease;
}

// 主按钮 - 使用主色
.btn-primary {
  background: var(--accent-blue);
  color: white;
  &:hover {
    background: var(--accent-blue-hover);
    transform: translateY(-1px);
  }
  &:disabled {
    background: var(--text-disabled);
    cursor: not-allowed;
    transform: none;
  }
}

// 次按钮 - 透明背景 + 边框
.btn-secondary {
  background: transparent;
  color: var(--text-primary);
  border: 1px solid rgba(255, 255, 255, 0.12);
  &:hover {
    background: var(--bg-hover);
    border-color: var(--accent-blue);
  }
}

// 成功按钮
.btn-success {
  background: var(--color-up);
  color: white;
  &:hover {
    background: #dc2626;
  }
}

// 危险按钮
.btn-danger {
  background: var(--color-down);
  color: white;
  &:hover {
    background: #059669;
  }
}
```

### 2.2 按钮尺寸

| 尺寸 | 类名 | 内边距 | 字号 |
|------|------|--------|------|
| 小 | `.btn-sm` | 6px 12px | 12px |
| 中 | `.btn` | 10px 20px | 14px |
| 大 | `.btn-lg` | 14px 28px | 16px |

### 2.3 Element Plus按钮


## 3. Tabs规范

### 3.1 el-tabs全局样式

```scss
// Tab基础样式 - 不变色
.el-tabs__item {
  transition: none !important;
  color: var(--text-secondary) !important;
  font-size: 14px;
  font-weight: 500;
  padding: 0 16px;
  height: 40px;
  line-height: 40px;
}

// Tab悬停 - 保持原色或轻微变化
.el-tabs__item:hover {
  color: var(--text-primary) !important;
}

// 激活Tab - 主色
.el-tabs__item.is-active {
  color: var(--accent-blue) !important;
  &:hover {
    color: var(--accent-blue) !important;
  }
}

// Tab下划线
.el-tabs__active-bar {
  background-color: var(--accent-blue) !important;
  height: 2px;
}
```

### 3.2 使用示例

```vue
<el-tabs v-model="activeTab" class="custom-tabs">
  <el-tab-pane label="因子分析" name="factor" />
  <el-tab-pane label="相关性分析" name="correlation" />
</el-tabs>
```

## 10. 使用检查清单

在开发新页面/组件时，请确保:

- [ ] 按钮使用 `.btn` + `.btn-primary` / `.btn-secondary` 类
- [ ] 标题使用 `<h1 class="page-title">` 或 `<h2 class="section-title">`，带SVG/FontAwesome图标
- [ ] Tab使用 `<el-tabs>` 并确保全局样式生效
- [ ] 下拉框使用 `<el-select>` 并确保样式覆盖
- [ ] 日期选择器使用 `<el-date-picker>`
- [ ] 计数器使用 `<el-input-number>`
- [ ] 卡片使用 `.card` 类
- [ ] 所有颜色使用CSS变量而非硬编码
