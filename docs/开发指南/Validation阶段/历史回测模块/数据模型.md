## 📊 数据表结构

### 1. 回测任务表 (backtest_task)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| task_id | VARCHAR(50) | 主键，任务ID |
| strategy_id | VARCHAR(100) | 策略ID |
| start_time | DATE | 回测开始时间 |
| end_time | DATE | 回测结束时间 |
| status | VARCHAR(20) | 状态 |
| result | JSON | 回测结果 |
| created_at | DATETIME | 创建时间 |

### 2. 回测结果表 (backtest_result)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| task_id | VARCHAR(50) | 任务ID |
| returns | DOUBLE | 收益率 |
| annual_return | DOUBLE | 年化收益 |
| sharpe | DOUBLE | 夏普比率 |
| max_drawdown | DOUBLE | 最大回撤 |
