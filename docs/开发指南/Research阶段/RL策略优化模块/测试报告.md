```python
class RealDataTradingEnv(gym.Env):
    """基于真实K线数据的交易环境"""

    def __init__(self, kline_df, initial_cash):
        self.kline_df = kline_df.reset_index(drop=True)
        self.position = 0  # 0=空仓, 1=持有
        self.current_cash = initial_cash

    def reset(self):
        """重置环境"""
        self.position = 0
        return self._get_state()

    def _get_state(self):
        """获取状态向量（6维）"""
        # 价格特征(2) + 持仓(1) + 现金(1) + 技术指标(2)
        state = np.zeros(6, dtype=np.float32)

    def step(self, action):
        """执行交易动作"""
        if action == 1:  # 买入
            self.position = 1
            reward = (next_price - current_price) / current_price
        elif action == 2:  # 卖出
            self.position = 0
            reward = (current_price - prev_price) / prev_price
```

**状态空间**:
- 价格特征: 当前价格、相对价格
- 持仓信息: 当前持仓
- 现金比例: 当前现金/初始现金
- 技术指标: MA5、MA20

**动作空间**:
- 0: 持有（不交易）
- 1: 买入
- 2: 卖出

**奖励函数**:
- 基于真实收益率计算
- 正确反映交易收益
- 惩罚无效交易

**回测逻辑**:
- 遍历历史K线数据
- 逐步执行交易决策
- 计算累计收益
- 支持长期回测
