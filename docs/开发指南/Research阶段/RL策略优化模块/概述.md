## 🎯 模块定位

### 重要说明
> 根据 **QLib 官方文档**，RL 模块主要用于优化 **Portfolio Strategy 的参数**（如 topk, n_drop）和 **订单执行策略**（如 TWAP/VWAP），**不是**用于优化因子组合！

### 在三阶段中的位置
- **Research阶段**: QLib高级模块，Portfolio Strategy参数优化 + 订单执行策略训练
- **核心价值**: 提供RL策略参数优化和订单执行策略训练能力

### 与其他模块的关系
- **上游**: ML训练模块 - 提供预测分数
- **核心**: Portfolio Strategy层 - RL优化策略参数
- **下游**: Validation阶段的RL策略验证模块

### QLib RL的真实用途

| 应用场景 | 说明 | QLib对应 |
|---------|------|---------|
| **Portfolio Strategy参数优化** | 优化TopkDropoutStrategy的topk/n_drop参数 | TopkDropoutStrategy + RL |
| **订单执行优化** | TWAP/VWAP/Implementation Shortfall | SingleOrderStrategy |
| **风险决策** | 动态止损、智能仓位调整 | RL-based风险控制 |

## 📊 核心功能

### 功能列表

#### 1. Portfolio Strategy参数优化 ⭐
- 优化 TopkDropoutStrategy 参数（topk, n_drop）
- 优化 EnhancedIndexingStrategy 参数
- RL环境构建（市场状态模拟）
- 奖励函数设计（Sharpe/Sortino等）

#### 2. 订单执行策略训练
- TWAP（时间加权平均价）策略
- VWAP（成交量加权平均价）策略
- Implementation Shortfall（执行缺口优化）
- **GPU加速训练** (RTX 4090等)
- **实时进度推送** (SSE)

#### 3. 策略优化
- 超参数调优
- 网络结构优化
- 奖励函数设计

#### 4. 回测评估
- RL策略回测
- 性能指标计算
- 策略对比

#### 5. 模型管理
- 模型保存/加载
- 模型版本管理
- 模型部署

### 输入输出

**输入**:
- 预测分数（来自ML模型）
- Portfolio Strategy配置（topk, n_drop等）
- 训练参数
- 奖励配置

**输出**:
- 优化后的策略参数
- 训练好的RL模型
- 策略性能报告
- 训练日志
