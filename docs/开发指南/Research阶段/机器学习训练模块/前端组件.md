## 🧩 组件详细设计

### 1. MLModelTrainingView - 模型训练页面

**文件路径**: `frontend/src/views/research/MLModelTrainingView.vue`

**功能特性**:
1. **模型配置区**（左侧面板）:
   - 模型类型选择（下拉框）
     - 传统ML: LGB, XGB, RF, Linear
     - 深度学习: LSTM, GRU, MLP
   - 任务类型选择（单选按钮）
     - 分类：股票涨跌预测
     - 回归：收益率预测
   - 数据源配置
     - 股票池选择（多选）
     - 时间范围选择（日期选择器）
     - 特征选择（从因子计算模块获取）
   - 标签配置
     - 标签类型（return/direction/volatility）
     - 预测horizon（滑块：1-60天）
   - 训练参数
     - 训练/验证/测试划分比例
     - Early stopping轮数
     - 随机种子

2. **模型参数配置区**（可折叠面板）:
   - LightGBM参数:
     - num_leaves, learning_rate, n_estimators
     - min_child_samples, subsample, colsample_bytree
   - XGBoost参数:
     - max_depth, eta, gamma, subsample
   - LSTM参数:
     - hidden_size, num_layers, dropout
     - sequence_length, batch_size
   - 超参数优化按钮
     - 链接到超参数优化页面

3. **训练监控区**（右侧面板）:
   - 实时训练进度条
   - 训练/验证损失曲线（ECharts）
   - 训练/验证指标变化
   - 预计剩余时间
   - 训练日志输出（滚动终端）

4. **训练结果区**（底部）:
   - 模型ID展示
   - 最终指标表格
   - 保存模型按钮
   - 导出模型按钮
   - 返回模型列表按钮

## 🎯 API集成

### 前端API封装

```typescript
// frontend/src/api/ml.ts

import { http } from './request'

export const mlAPI = {
  // 训练模型
  trainModel: (config: MLTrainingConfig) => {
    return http.post('/v1/research/ml/train', config)
  },

  // 预测
  predict: (modelId: string, data: PredictionRequest) => {
    return http.post(`/v1/research/ml/predict`, { model_id: modelId, ...data })
  },

  // 获取模型列表
  getModels: (params?: QueryParams) => {
    return http.get('/v1/research/ml/models', { params })
  },

  // 获取模型详情
  getModelDetails: (modelId: string) => {
    return http.get(`/v1/research/ml/models/${modelId}`)
  },

  // 评估模型
  evaluateModel: (modelId: string, data: EvaluationRequest) => {
    return http.post(`/v1/research/ml/models/${modelId}/evaluate`, data)
  },

  // 超参数优化
  optimizeHyperparameters: (config: OptimizationRequest) => {
    return http.post('/v1/research/ml/optimize', config)
  },

  // 获取特征重要性
  getFeatureImportance: (modelId: string, topN?: number) => {
    return http.get(`/v1/research/ml/features?model_id=${modelId}&top_n=${topN || ''}`)
  }
}
```
