# 分钟线数据验证报告

> 验证日期: 2026-02-05
> 验证目标: 确认分钟线数据的正确获取方式

---

## 问题背景

之前的测试中，分钟线数据一直返回0条。通过对比adapter实现，发现是参数使用不当导致。

---

## 关键发现

### 发现1: start_time和end_time参数使用 ✅

**三种有效方式**（来自[01-参数实测报告.md](./01-参数实测报告.md)）：

```python
# 方式1: 完全不传参数（推荐）
data = xtdata.get_market_data_ex(
    period='5m',
    count=120
)

# 方式2: 传空字符串 ''
data = xtdata.get_market_data_ex(
    period='5m',
    start_time='',
    end_time='',
    count=120
)

# 方式3: 传 0
data = xtdata.get_market_data_ex(
    period='5m',
    start_time=0,
    end_time=0,
    count=120
)
```

**结论**: 三种方式都有效，官方文档建议**不传参数**（缺省时用count）。

---

### 发现1.5: 分钟线数据获取限制 ✅

**重要发现**（来自[103-参数完整研究.md](./103-参数完整研究.md)）：

| 方式 | 限制 | 说明 |
|------|------|------|
| **在线获取** | 22.6个交易日（1083条5分钟K线） | `get_market_data_ex(count=1500)` |
| **下载+读取** | 几乎无限制 | `download_history_data()` + `get_market_data_ex()` |

**计算过程**:
- 5分钟K线一天: 48条（4小时 × 12条/小时）
- 1083条 ÷ 48条/天 = **22.6个交易日**

**使用建议**:
- 在线获取适合: 临时查询、最近数据
- 下载+读取适合: 历史数据、回测系统

---

### 发现2: 必须先订阅再获取

**正确流程**：
```python
# 1. 先订阅
xtdata.subscribe_quote(
    stock_code='600519.SH',
    period='5m',
    count=0
)

# 2. 再获取
data = xtdata.get_market_data_ex(
    field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
    stock_list=['600519.SH'],
    period='5m',
    start_time='',
    end_time='',
    count=120,
    dividend_type='none',
    fill_data=True
)
```

---

### 发现3: dividend_type必须是字符串

之前已验证：
- ✅ `'none'` - 不复权
- ✅ `'front'` - 前复权
- ✅ `'back'` - 后复权
- ❌ `0, 1, 2` - 数字会报错

## 最佳实践总结

### 获取分钟线数据的标准流程

```python
def get_minute_kline(symbol, period='5m', count=120):
    """
    获取分钟K线（标准流程）

    Args:
        symbol: 股票代码
        period: 周期 ('1m', '5m', '15m', '30m', '60m')
        count: 条数

    Returns:
        DataFrame or None
    """
    from xtquant import xtdata

    try:
        # 1. 先订阅
        xtdata.subscribe_quote(
            stock_code=symbol,
            period=period,
            count=0
        )

        # 2. 再获取（使用正确的参数）
        data = xtdata.get_market_data_ex(
            field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
            stock_list=[symbol],
            period=period,
            start_time='',  # ⭐ 必须用空字符串
            end_time='',    # ⭐ 必须用空字符串
            count=count,
            dividend_type='none',  # ⭐ 必须用字符串
            fill_data=True
        )

        if data and symbol in data:
            return data[symbol]
        else:
            return None

    except Exception as e:
        logger.error(f"获取分钟K线失败: {e}")
        return None
```
