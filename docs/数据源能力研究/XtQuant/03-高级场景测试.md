**测试步骤**:
1. 本地只有部分历史数据
2. 请求比本地更多的数据
3. 验证是否自动补全
4. 验证补全后的数据完整性

**关键验证**:
```python
# 请求100条

# 本地只有80条
local_count = 80

# 期望：自动在线获取缺失的20条
expected_total = 100

# 验证
assert len(result) == expected_total
assert result[0].date == expected_start_date
assert result[-1].date == expected_end_date
```

**业务需求**:

- 用户请求前复权数据

**测试验证**:

#### 场景3.3: 分时线与K线对应关系

**业务需求**:
- 验证分时线聚合后与K线一致
- 1分钟K线聚合后 = 日K线

**测试验证**:
```python

    stock_list=['600519.SH'],
    period='1m',
    count=240
)

# 聚合为日K线
day_kline_from_minute = aggregate_to_day(minute_kline)

# 获取日K线

    stock_list=['600519.SH'],
    period='1d',
    count=1
)

# 验证
assert day_kline_from_minute['open'].iloc[-1] == day_kline['open'].iloc[-1]
assert day_kline_from_minute['close'].iloc[-1] == day_kline['close'].iloc[-1]
assert day_kline_from_minute['high'].iloc[-1] == day_kline['high'].iloc[-1]
assert day_kline_from_minute['low'].iloc[-1] == day_kline['low'].iloc[-1]
```

#### 场景5.3: 批量股票监控

**业务需求**:
- 监控自选股列表（如100只）
- 实时获取最新价格
- 生成涨跌排名

**测试验证**:
```python
# 批量订阅
watchlist = [f"60{i:04d}.SH" for i in range(100)]

for stock in watchlist:
    subscribe_quote(stock, period='1d')

# 批量获取（每隔3秒）
while True:
    quotes = get_full_tick(watchlist)

    # 计算涨跌幅
    for stock, data in quotes.items():
        change_pct = (data['lastPrice'] - data['lastClose']) / data['lastClose'] * 100
        data['change_pct'] = change_pct

    # 排序
    sorted_quotes = sorted(quotes.items(), key=lambda x: x[1]['change_pct'], reverse=True)

    # 显示前10
    for stock, data in sorted_quotes[:10]:
        print(f"{stock}: {data['change_pct']:.2f}%")

    time.sleep(3)
```
