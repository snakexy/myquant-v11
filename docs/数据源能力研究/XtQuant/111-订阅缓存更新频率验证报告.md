## 关键发现

### 1. 性能特性

**单只股票获取速度**:
- 订阅后: ~0.5ms
- **比100只股票快13倍** (100只 = 6.5ms, 单只 = 0.5ms)
- **极度稳定**: 波动范围 < 0.03ms

**对比其他操作**:
- 订阅缓存读取: 0.5ms
- 100只股票批量读取: 6.5ms
- 本地K线读取: 6-8ms
- 在线K线获取: ~500ms

---

### 2. 数据行为

**交易时间** (推断):
- 数据更新频率: 约3秒
- 每次get_full_tick()获取最新快照
- 价格和时间戳会实时变化

**非交易时间** (已验证):
- 返回最后收盘价
- 时间戳为最后收盘时间
- 数据稳定不变

---

### 3. 缓存机制

**发现**:
- 订阅缓存 ≠ 本地文件缓存
- 两个QMT实例共享数据目录后，所有数据都变成了"缓存"
- get_full_tick() 优先从共享数据目录读取

**性能对比**:
```
未配置共享数据目录前:
  订阅股票: 0.5ms (内存缓存)
  未订阅: ~50ms (网络获取)

配置共享数据目录后:
  订阅股票: 0.5ms (本地缓存)
  未订阅: 0.5ms (本地缓存)
```

### 场景1: 实时行情列表

```python
class RealtimeQuoteList:
    """实时行情列表"""

    def __init__(self):
        self.symbols = []

    def add_symbols(self, symbols):
        """添加股票（自动订阅）"""
        for symbol in symbols:
            xtdata.subscribe_quote(symbol, period='1d')
            self.symbols.append(symbol)

    def refresh(self):
        """刷新行情（3秒一次）"""
        while True:
            # 批量获取（超快！）
            quotes = xtdata.get_full_tick(self.symbols)
            # 100只股票 ~6.5ms
            # 1只股票 ~0.5ms

            # 推送前端
            push_to_frontend(quotes)

            time.sleep(3)  # 3秒刷新
```

**性能**:

- 300只股票: ~20ms

### 场景3: 非交易时间显示

```python
def is_trading_time():
    """判断是否交易时间"""
    from datetime import datetime
    now = datetime.now()

    # 周末
    if now.weekday() >= 5:
        return False

    # 时间: 9:30-15:00
    hour, minute = now.hour, now.minute
    time_min = hour * 60 + minute
    return 570 <= time_min <= 900  # 9:30-15:00

def get_quotes_smart(symbols):
    """智能获取行情"""

    if is_trading_time():
        # 交易时间: 实时数据
        return xtdata.get_full_tick(symbols)
    else:
        # 非交易时间: 也是实时数据（最后收盘价）
        # XtQuant自动处理，无需特殊逻辑
        return xtdata.get_full_tick(symbols)
```

**优势**: XtQuant已经自动处理了非交易时间的情况！
