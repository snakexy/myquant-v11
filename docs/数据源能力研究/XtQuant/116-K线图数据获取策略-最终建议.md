## 🚀 最终推荐策略

### 方案A: 纯在线获取（推荐）⭐⭐⭐⭐⭐

**适用于所有K线图场景**

```python
def get_kline_data(symbol, period, count):
    """
    获取K线数据（纯在线获取方案）

    适用场景:
    - 所有K线周期（日K、1h、30m、15m、5m、1m）
    - 所有时间范围（1周到1年）
    - 所有数据量（10条到6000+条）

    性能:
    - 平均耗时: 9-36ms
    - 最快: 9ms
    - 最慢: 36ms
    """
    from xtquant import xtdata

    # 1. 订阅
    xtdata.subscribe_quote(symbol, period=period, count=0)

    # 2. 在线获取
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time='',
        end_time='',
        count=count,
        dividend_type='none'
    )

    return data[symbol] if data and symbol in data else None
```

**优点**:
- ✅ 代码极简（一次API调用）
- ✅ 性能优秀（9-36ms）
- ✅ 数据实时
- ✅ 无需管理本地缓存
- ✅ 支持所有K线周期和数据量

**缺点**:
- ⚠️ 可能存在未知的上限（但测试已到6336条）

### 方案B: 混合策略（保守）⭐⭐⭐

**先在线获取，失败则下载+读取**

```python
def get_kline_data_hybrid(symbol, period, count):
    """
    获取K线数据（混合策略）

    策略:
    1. 优先使用在线获取（快速）
    2. 如果在线获取失败或数据不足，使用下载+读取
    """
    from xtquant import xtdata
    from datetime import datetime, timedelta

    # 1. 先尝试在线获取
    xtdata.subscribe_quote(symbol, period=period, count=0)

    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time='',
        end_time='',
        count=count,
        dividend_type='none'
    )

    # 2. 检查数据是否足够
    if data and symbol in data and len(data[symbol]) >= max(count * 0.9, 1):
        return data[symbol]

    # 3. 数据不足，使用下载+读取
    days_needed = estimate_days(period, count)
    end_date = datetime.now().strftime('%Y%m%d')
    start_date = (datetime.now() - timedelta(days=days_needed)).strftime('%Y%m%d')

    # 下载
    xtdata.download_history_data(
        stock_code=symbol,
        period=period,
        start_time=start_date,
        end_date=end_date
    )

    # 读取
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time=int(start_date),
        end_time=int(end_date),
        dividend_type='none'
    )

    return data[symbol] if data and symbol in data else None
```

**优点**:
- ✅ 有容错机制
- ✅ 支持超大数据量

**缺点**:
- ❌ 代码复杂
- ❌ 下载+读取性能差（100-300ms）

## 📝 实施建议

### 1. K线图API实现

```python
# backend/api/v1/market.py

@router.get("/kline")
async def get_kline_data(
    symbol: str,
    period: str,  # '1d', '1h', '30m', '15m', '5m', '1m'
    count: int = Query(default=500, ge=1, le=10000)
):
    """
    获取K线数据

    参数:
    - symbol: 股票代码
    - period: K线周期 ('1d', '1h', '30m', '15m', '5m', '1m')
    - count: K线数量（1-10000）

    返回:
    - K线数据（时间、开高低收、成交量）
    """
    import xtdata
    import time

    start = time.time()

    try:
        # 订阅
        xtdata.subscribe_quote(symbol, period=period, count=0)

        # 在线获取
        data = xtdata.get_market_data_ex(
            field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
            stock_list=[symbol],
            period=period,
            start_time='',
            end_time='',
            count=count,
            dividend_type='none'
        )

        elapsed = (time.time() - start) * 1000

        if data and symbol in data:
            df = data[symbol]

            return {
                'code': 200,
                'data': df.to_dict('records'),
                'count': len(df),
                'period': period,
                'elapsed_ms': elapsed
            }
        else:
            return {
                'code': 404,
                'message': 'No data available'
            }

    except Exception as e:
        return {
            'code': 500,
            'message': str(e)
        }
```

### 2. period参数映射

```python
# 前端可能发送的参数，需要映射到XtQuant格式
PERIOD_MAP = {
    'day': '1d',
    'week': '1w',
    '1h': '1h',
    '60m': '1h',  # ⭐ 映射60m到1h
    '30m': '30m',
    '15m': '15m',
    '5m': '5m',
    '1m': '1m',
    'minute': '1m',
}

def normalize_period(period: str) -> str:
    """标准化period参数"""
    return PERIOD_MAP.get(period, period)
```

### 3. 前端调用示例

```typescript
// frontend/src/api/market.ts

export async function getKlineData(
  symbol: string,
  period: '1d' | '1h' | '30m' | '15m' | '5m' | '1m',
  count: number = 500
) {
  const result = await http.get('/api/v1/market/kline', {
    params: { symbol, period, count }
  });

  if (result.data.code === 200) {
    const klineData = result.data.data;

    return klineData;
  }

  throw new Error(result.data.message);
}
```

1. **在线获取适用于所有K线图场景** ⭐⭐⭐⭐⭐

   - 优先使用纯在线获取方案

### 与订阅缓存的关系

| 数据获取方式 | 性能 | 适用场景 |
|------------|------|---------|
| **订阅缓存** | 0.5ms | 实时快照（已订阅股票） |
| **在线获取** | 9-36ms | K线历史数据 |
| **下载+读取** | 100-300ms | 历史回测（暂不推荐） |

**建议**: K线图使用在线获取，行情列表使用订阅缓存
