## 问题

**参数格式错误！**

| 参数格式 | 结果 |
|---------|------|

### 方案2: 在线获取（推荐用于实时数据）

```python
def get_realtime_minute_kline(symbol, period='5m', count=120):
    """
    获取实时分钟K线（在线获取）

    适用于：获取最近N条实时数据

    限制：
    - 1分钟K线: 最多1261条（约21小时）
    - 5分钟K线: 最多678条（约56.5小时）

    Args:
        symbol: 股票代码
        period: 周期 '1m', '5m', '15m', '30m', '60m'
        count: 条数

    Returns:
        DataFrame or None
    """
    from xtquant import xtdata

    try:
        # 1. 先订阅
        xtdata.subscribe_quote(symbol, period=period, count=0)

        # 2. 再获取
        data = xtdata.get_market_data_ex(
            field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
            stock_list=[symbol],
            period=period,
            start_time='',  # ⭐ 空字符串
            end_time='',    # ⭐ 空字符串
            count=count,
            dividend_type='none',
            fill_data=True
        )

        if data and symbol in data:
            return data[symbol]
        else:
            return None

    except Exception as e:
        logger.error(f"获取实时分钟K线失败: {e}")
        return None
```

## 最终结论

### ✅ 好消息

1. **分钟线支持下载** - 可以获取任意时间范围的数据
2. **下载速度很快** - 7天数据只需0.02秒
3. **两种方式都可用** - 下载+读取（历史）+ 在线获取（实时）

### ⚠️ 注意事项

1. **参数格式严格** - 字符串 vs 整数 vs 空字符串
2. **在线获取限制** - 最多2-3天数据（678条5分钟K线）
3. **首次调用开销** - 760ms初始化开销

### 📋 使用建议

| 场景 | 推荐方式 | 参数 |
