## API签名

```python
xtdata.get_financial_data(
    stock_list,      # 股票列表
    table_list=[],    # 报表列表
    start_time='',    # 开始时间 (格式: '20240101')
    end_time='',      # 结束时间 (格式: '20241231')
    report_type='report_time'  # 'announce_time' 或 'report_time'
)

# 返回:
{
    'field': list[str],   # 财务字段列表
    'date': list[int],    # 报告期列表
    'stock': list[str],   # 股票列表
    'value': list[list[float]]  # 数值 (行=字段, 列=报告期)
}
```

## 分析

### 可能原因

1. **需要VIP权限** ⭐ 最可能
   - 财务数据属于高级数据
   - 需要付费订阅
   - 非VIP账户无法访问

2. **QMT客户端未下载**
   - 财务数据需要手动下载
   - 需要在QMT客户端中更新财务数据

3. **时间范围无数据**

   - 尝试其他时间范围

4. **账户限制**
   - 当前账户可能没有财务数据权限

## 常见的财务数据替代方案

### Tushare (免费额度)

```python
import tushare as ts

# 需要注册获取token
ts.set_token('your_token')
pro = ts.pro_api()

# 获取资产负债表
df = pro.balancesheet(
    ts_code='600519.SH',
    period='20240930',
    report_type='1'
)
```

**优点**:
- 免费额度（每日200次）
- 数据质量高

**缺点**:
- 免费额度有限
- 需要注册

---

### AKShare (完全免费)

```python
import akshare as ak

# 获取资产负债表
df = ak.stock_balance_sheet_by_yearly(symbol="600519")

# 获取利润表
df = ak.stock_profit_sheet_by_yearly(symbol="600519")
```

**优点**:
- 完全免费
- 无需注册
- 数据全面

**缺点**:
- 数据质量一般
- 更新不及时
- 可能不稳定
