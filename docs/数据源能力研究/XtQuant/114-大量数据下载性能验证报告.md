## 优化建议

### 1. 批量下载策略

**推荐方案**: 分批下载
```python
def download_all_stocks():
    """分批下载全市场数据"""

    # 获取全A股
    all_stocks = xtdata.get_stock_list_in_sector('沪深A股')

    # 分批：每批500只
    batch_size = 500
    for i in range(0, len(all_stocks), batch_size):
        batch = all_stocks[i:i+batch_size]

        # 下载当前批次
        for stock in batch:
            xtdata.download_history_data(
                stock_code=stock,
                period='1d',
                start_time='20240101',
                end_time='20241231'
            )

        print(f"已完成: {i+len(batch)}/{len(all_stocks)}")
```

**优势**:
- 可控的进度
- 失败易重试
- 内存占用小

---

### 2. 并发下载策略

**推荐方案**: 4-8线程
```python
import threading

def download_concurrent(symbols, num_threads=4):
    """多线程并发下载"""

    def worker(chunk):
        for symbol in chunk:
            xtdata.download_history_data(
                stock_code=symbol,
                period='1d',
                start_time='20240101',
                end_time='20241231'
            )

    # 分割任务
    chunk_size = len(symbols) // num_threads
    chunks = [symbols[i:i+chunk_size]
              for i in range(0, len(symbols), chunk_size)]

    # 启动线程
    threads = []
    for chunk in chunks:
        thread = threading.Thread(target=worker, args=(chunk,))
        threads.append(thread)
        thread.start()

    # 等待完成
    for thread in threads:
        thread.join()
```

**优势**:
- 速度提升16%
- 适合大规模下载
