## ⚠️ 问题与限制

### 1. 实时模式问题
**问题描述**: TQ数据源初始化失败
```
RuntimeError: TQ数据源登录初始化失败
```

**原因分析**:
- TQ数据源可能需要账号登录
- TQ数据源服务器可能不可用
- 连接参数配置可能不正确

**影响**: 无法测试真实的TdxQuant数据

**建议**:
1. 配置正确的TQ数据源连接参数
2. 或者使用本地TdxQuant数据文件
3. 或者使用其他数据源进行对比测试

### 2. L0订阅功能
**问题描述**: 适配器中没有明确的订阅推送API

**建议**:
- 检查TdxQuant SDK是否支持订阅功能
- 如果支持，添加到适配器
- 如果不支持，文档中标注"不支持"是正确的
