## 🔑 关键发现

### 发现1: 参数类型至关重要
- `dividend_type` 必须是字符串，不能用数字
- 文档: [06-测试结果对比报告.md](06-测试结果对比报告.md)
- 修复: [07-Bug修复报告.md](07-Bug修复报告.md)

### 发现2: 分钟线实际限制2-3天（非16天）
- 1分钟K线: 最多1261条（约21小时）
- 5分钟K线: 最多678条（约56.5小时 = 2.3天）
- 文档: [15-16天限制验证报告-最终版.md](15-16天限制验证报告-最终版.md) ⭐⭐

### 发现3: 下载+读取比纯在线快3倍
- 下载（50ms）+ 读取（100ms）= 150ms
- 在线获取: 500ms
- 文档: [08-最佳实践方案.md](08-最佳实践方案.md)

### 发现4: 订阅缓存性能卓越
- 单只股票: 0.5ms
- 批量获取: 0.17ms/只
- 文档: [01-参数实测报告.md](01-参数实测报告.md)

### 发现5: 分钟线首次调用有760ms开销
- 首次调用: 760ms
- 后续调用: 6-8ms
- 文档: [14-分钟线验证报告.md](14-分钟线验证报告.md)
