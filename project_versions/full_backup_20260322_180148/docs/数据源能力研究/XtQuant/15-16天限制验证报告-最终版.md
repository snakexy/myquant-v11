## 关键结论

### ✅ 确认的事实

1. **不存在16天限制**
   - 官方文档说的"16天限制"可能是误传或者适用于其他场景

2. **实际限制约5-14个交易日（包含集合竞价）**
   - 1分钟K线: 约5个交易日（1261条）
   - 5分钟K线: 约13.5个交易日（678条）

3. **只能使用count获取**
   - ✅ `start_time='', end_time='', count=678` - 有效
   - ❌ `start_time=20240101, end_time=20241231, count=0` - 无效

4. **首次调用有额外开销**
   - 1分钟K线首次: 766ms
   - 后续调用: 约5-15ms

## 获取分钟线的正确方式

### 方案1: 使用count参数（推荐）

```python
def get_minute_kline_limit(symbol, period='5m'):
    """
    获取分钟K线（使用count参数）

    限制（在线获取）：
    - 1分钟K线: 最多1261条（约5.25个交易日）
    - 5分钟K线: 最多678条（约14.1个交易日）
    - 15分钟K线: 最多226条（约14.1个交易日）
    - 30分钟K线: 最多113条（约14.1个交易日）
    - 60分钟K线: 最多56条（约14个交易日）
    """
    from xtquant import xtdata

    # 确定最大count
    # 包含集合竞价时间（9:15-15:00），每天250分钟
    max_count = {
        '1m': 1261,  # 约5个交易日 (1261÷250)
        '5m': 678,   # 约13.5个交易日 (678÷50)
        '15m': 226,  # 约14个交易日 (226÷16≈14.1)
        '30m': 113,  # 约14个交易日 (113÷8≈14.1)
        '60m': 56,   # 约14个交易日 (56÷4=14)
    }

    count = max_count.get(period, 500)

    # 先订阅
    xtdata.subscribe_quote(symbol, period=period, count=0)

    # 再获取
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time='',  # 必须用空字符串
        end_time='',    # 必须用空字符串
        count=count,
        dividend_type='none',  # 必须用字符串
        fill_data=True
    )

    return data.get(symbol)
```

### 方案2: 下载+读取（获取更长时间数据）⭐

```python
def get_minute_kline_long_history(symbol, period='5m', start_date='20240101', end_date='20241231'):
    """
    下载+读取方式获取历史分钟K线（推荐）

    优势：
    1. 几乎无时间限制
    2. 数据完整可靠
    3. 适合历史回测

    策略：
    1. 使用download_history_data下载到本地
    2. 使用get_market_data_ex读取本地数据
    """
    from xtquant import xtdata

    # 1. 下载历史数据到本地
    xtdata.download_history_data(
        stock_code=symbol,
        period=period,
        start_time=start_date,  # 字符串格式 '20240101'
        end_time=end_date       # 字符串格式 '20241231'
    )

    # 2. 读取本地数据
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time=start_date,
        end_time=end_date,
        dividend_type='none'
    )

    return data.get(symbol)
```
