```python
def get_recent_minute_kline(symbol, period='5m'):
    """
    获取最近3-4天分钟K线

    限制:
    - 1分钟K线: 最多约21小时 (1261条)
    - 5分钟K线: 最多约22.6个交易日 (1083条)
    """
    from xtquant import xtdata

    # 确定最大count
    max_count = {
        '1m': 1261,
        '5m': 1083,   # ⭐ 实际上限
        '15m': 361,
        '30m': 180,
        '60m': 90,
    }

    count = max_count.get(period, 500)

    # 先订阅
    xtdata.subscribe_quote(symbol, period=period, count=0)

    # 再获取
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time='',
        end_time='',
        count=count,
        dividend_type='none',
        fill_data=True
    )

    return data.get(symbol)
```

## 对官方文档的质疑

### 官方说法（可能）
> "分钟线数据最多获取16天"

### 实际测试结果
> **分钟线数据最多获取约22.6个交易日（1083条5分钟K线）**

### 可能的解释

1. **16天可能是总容量**
   - 本地存储最多保存16天数据
   - 但在线获取只有3-4天

2. **不同版本限制不同**
   - XtQuant不同版本可能有不同的限制
   - 当前版本可能限制了在线获取

3. **需要付费升级**
   - 16天限制可能是付费版功能
   - 免费版只有3-4天

4. **文档过时**
   - 官方文档未更新
   - 实际限制已改变
