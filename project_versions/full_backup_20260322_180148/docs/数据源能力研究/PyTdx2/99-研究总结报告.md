## 复权功能研究

### 除权除息 API

`get_xdxr_info(market, code)` 返回16个字段：

```python
{
    'year': 2025,
    'month': 7,
    'day': 10,
    'category': 1,           # 1=除权除息
    'name': '除权除息',
    'fenhong': 4.10,         # 分红（元/10股）
    'songzhuangu': 0.0,      # 送股（股/10股）
    'peigu': 0.0,            # 配股（股/10股）
    'peigujia': 0.0,         # 配股价（元/股）
    ...
}
```

### 前复权公式

```
除权后理论价格 = (原收盘价 + 配股比例 × 配股价 - 每股分红) / (1 + 送股比例 + 配股比例)

复权因子 = 除权后价格 / 除权前价格
```

### 测试结果

- 600000: 26条除权除息记录
- 000001: 31条除权除息记录
- 600036: 28条除权除息记录
- 000002: 36条除权除息记录

## 优势与劣势

### 优势 ⭐

1. **无依赖** - 不需要安装其他软件
2. **24/7可用** - 不受交易时间限制
3. **完整数据** - K线/财务/板块全覆盖
4. **前复权** - 完整的除权除息计算
5. **稳定可靠** - 服务器列表广泛

### 劣势 ⚠️

1. **性能中等** - 比TdxQuant慢11倍，比XtQuant慢约20倍
2. **无订阅推送** - 轮询模式，无法实现真正实时
3. **K线限制** - 每次最多800条
4. **备用定位** - 作为TdxQuant/XtQuant之后的备用

## 结论

### 核心优势：24/7可用（TdxQuant无法替代）

| 数据源 | 交易时间可用 | 非交易时间可用 | 说明 |
|--------|-------------|---------------|------|
| **PyTdx2** | ✅ | ✅ | 24/7在线，无限制 |
| TdxQuant | ✅ | ❌ | **非交易时间无法获取最新数据** |
| XtQuant | ✅ | ✅ | 盘后可用 |
| QLib | ✅ | ✅ | 本地数据 |

### 定位

| 时间段 | 数据源优先级 | 说明 |
|--------|-------------|------|
| **交易时间** | TdxQuant(0.60ms) → XtQuant(0.90ms) → PyTdx2(10ms) | TdxQuant最快 |
| **非交易时间** | XtQuant(0.90ms) → PyTdx2(10ms) | TdxQuant不可用 |
| **离线环境** | PyTdx2 ✅ | 无XtQuant/TdxQuant |

### 适用场景

1. **离线环境** - 无MiniQMT/通达信时唯一选择
2. **备用数据源** - XtQuant不可用时
3. **24/7数据获取** - 作为XtQuant的补充
