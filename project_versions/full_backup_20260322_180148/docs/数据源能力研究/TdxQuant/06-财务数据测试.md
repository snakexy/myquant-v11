> 测试日期: 2026-03-04

## 概述

TdxQuant在财务数据方面具有明显优势，支持GP1-GP46共63个财务指标字段。

## 财务数据能力

### GP1-GP46财务指标

TdxQuant `get_stock_info()` 函数返回63个字段，包含完整的财务数据。

### 财务指标分类

#### 1. 基本信息
- Name: 公司名称 ✅
- ActiveCapital: 活跃股本
- J_zgb: 总股本
- J_zzc: 总资产

#### 2. 盈利能力指标
- J_yysy: 营业收益
- J_tzsy: 投资收益
- J_jllr: 净利润
- J_xsmllr: 销售毛利率
- J_xsjllr: 销售净利率
- J_zyl: 总资产收益率
- J_jzcjllr: 净资产收益率

#### 3. 偿债能力指标
- J_zcfz: 资产负债率
- J_ldzc: 流动比率
- J_ldzc: 速动比率

#### 4. 成长性指标
- J_yysyzz: 营业收入增长
- J_jllzz: 净利润增长
- J_zczz: 总资产增长

#### 5. 其他指标
- BelongHS300: 是否属于沪深300
- J_gpr: 市盈率
- J_hpr: 市净率
- ... (共63个字段)

### 动态估值指标

TdxQuant `get_more_info()` 函数返回86个字段，包含动态估值指标。

#### 估值指标
- DynaPE: 动态市盈率
- StaticPE_TTM: 静态市盈率
- PB_MRQ: 市净率
- DYRatio: 股息率

#### 交易指标
- ZAF: 振幅
- Zjl: 换手率
- HisHigh: 历史最高价
- HisLow: 历史最低价
- ZTPrice: 涨停价
- DTPrice: 跌停价

#### 技术指标
- BetaValue: 贝塔系数
- ... (共86个字段)

## 财务数据使用场景

### 1. 基本面分析

**场景**: 需要分析股票的基本面情况

**推荐数据源**: **TdxQuant**（63个财务指标）

**使用方式**:
```python
# 获取股票财务数据
financial_info = tdxquant.get_stock_info('600519.SH')

# 基本面分析
pe_ratio = financial_info['J_gpr']           # 市盈率
pb_ratio = financial_info['J_hpr']           # 市净率
roe = financial_info['J_jzcjllr']            # 净资产收益率
debt_ratio = financial_info['J_zcfz']         # 资产负债率

# 价值投资筛选
if (pe_ratio < 20 and pb_ratio < 2 and roe > 15):
    print("价值投资标的")
```

### 2. 估值分析

**场景**: 需要分析股票的估值水平

**推荐数据源**: **TdxQuant**（86个估值指标）

**使用方式**:
```python
# 获取股票估值数据
valuation_info = tdxquant.get_more_info('600519.SH')

# 估值分析
dynamic_pe = valuation_info['DynaPE']         # 动态市盈率
static_pe = valuation_info['StaticPE_TTM']     # 静态市盈率
pb = valuation_info['PB_MRQ']                 # 市净率
dividend_yield = valuation_info['DYRatio']      # 股息率

# 估值筛选
if (dynamic_pe < 15 and static_pe < 20 and pb < 2):
    print("低估标的")
```

### 3. 成长性分析

**场景**: 需要分析股票的成长性

**推荐数据源**: **TdxQuant**（完整成长性指标）

**使用方式**:
```python
# 获取成长性数据
growth_info = tdxquant.get_stock_info('600519.SH')

# 成长性分析
revenue_growth = growth_info['J_yysyzz']       # 营业收入增长
profit_growth = growth_info['J_jllzz']          # 净利润增长
asset_growth = growth_info['J_zczz']           # 总资产增长

# 成长性筛选
if (revenue_growth > 10 and profit_growth > 10):
    print("高成长标的")
```
