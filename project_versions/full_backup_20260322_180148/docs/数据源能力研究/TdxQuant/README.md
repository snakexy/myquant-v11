# TdxQuant 数据源能力研究

## 概述

TdxQuant（通达信量化接口）是通达信提供的专业量化数据接口，基于通达信客户端的API封装。

## 核心优势

### 1. 实时性能领先
- **L1实时快照**: 0.60ms（所有数据源中最快）
- **批量获取**: 1.61ms/股（批量性能良好）
- **24/7可用**: 全天候可用，不受交易时间限制

### 2. 板块数据优势
- **板块指数**: 行业板块、概念板块等完整的板块指数数据
- **成分股列表**: 各板块的成分股列表
- **板块实时行情**: 板块指数的实时快照

### 3. 盘中辅助信息丰富
- **资金流向**: 实时资金流向数据
- **买卖盘口**: 五档盘口数据
- **实时分笔**: 逐笔成交数据

## 主要限制

### 1. 实时数据限制
- ⚠️ **仅快照数据**: 不支持实时推送，只能获取当前快照
- ⚠️ **订阅功能未验证**: 虽然知道可以订阅，但推送机制不清楚
- ⚠️ **不是完整序列**: 快照只是当前时间点的数据，不是历史序列

### 2. 历史数据限制
- ⚠️ **不支持当天分钟数据**: 只能获取历史分钟数据（昨天及以前的，历史分钟数据**是支持的**）
- ⚠️ **K线数据量受限**: 最多1088根日线/周线/月线/1分钟/5分钟数据
- ⚠️ **15分钟/30分钟/60分钟**: 最多1000根

### 3. 其他限制
- ⚠️ **订阅上限**: 300只/实例（但功能未验证）
- ⚠️ **数据格式**: 返回格式可能需要转换处理

## 适用场景

### 最佳适用场景 ✅
1. **股票列表查询** - 5,489只股票全覆盖（主要优势）
2. **板块数据查询** - 586个板块完整，板块指数、成分股等（主要优势）
3. **股票名称获取** - 中文名称显示（主要优势）
4. **板块选股功能** - 支持基于板块的选股（主要优势）
5. **盘中辅助信息获取** - 资金流向、盘口、分笔等
6. **实时快照获取** - 0.60ms超快速度
7. **历史K线补充** - 非当天历史K线数据
8. **盘后数据更新** - 24/7可用

### 不适用场景 ❌
1. **当天分钟数据** - 不支持当天（今天）的分钟数据（历史分钟数据是支持的）
2. **实时推送订阅** - 功能未验证
3. **超长K线数据** - 最多1088根
4. **高频推送监控** - 无推送机制

## 数据类型支持

### L0-L5层级支持

| 层级 | 说明 | 支持情况 | 备注 |
|------|------|----------|------|
| L0 | 订阅推送 | ❌ 不支持 | 订阅功能未验证 |
| L1 | 实时快照 | ✅ 支持 | **0.60ms，最快** |
| L2 | 历史快照 | ✅ 支持 | 但不支持当天分钟数据 |

| L4 | 财务数据 | ✅ 支持 | GP1-GP46完整支持 |
| L5 | 板块管理 | ✅ 支持 | **板块数据是主要优势** |

## 优先级配置

根据实际测试结果，TdxQuant在MyQuant系统中的优先级：

### 盘中实时优先级
- **优先级1**: XtQuant订阅（<1ms，真正的实时推送，用于L0订阅缓存）
- **优先级1**: TdxQuant实时快照（0.60ms，**最快**）+ 板块数据优势
- **优先级2**: XtQuant批量（0.90ms）
- **优先级3**: PyTdx（14ms）

### 历史数据优先级
- **优先级1**: QLib本地DB（7-10ms）
- **优先级2**: TdxQuant（24/7，板块数据优势）
- **优先级2**: XtQuant（0.90ms）
- **优先级3**: PyTdx（14ms）

## 总结

TdxQuant的核心优势在于：
1. **L1实时快照最快** - 0.60ms是所有数据源中最快的
2. **股票列表完整** - 5,489只股票全覆盖
3. **板块数据完整** - 586个板块、板块指数、成分股等是主要强项
4. **股票名称支持** - 中文名称显示，用户体验友好
5. **盘中辅助信息丰富** - 资金流向、盘口、分笔等
6. **24/7可用** - 不受交易时间限制

但需要注意：
1. **仅快照数据** - 不支持实时推送
2. **不支持当天分钟数据** - 历史分钟数据有限制
3. **K线数据量受限** - 最多1088根

因此，TdxQuant的最佳使用场景是：
- 股票列表查询（5,489只全覆盖）
- 板块数据查询（586个板块完整，板块指数、成分股等，主要优势）
- 股票名称显示（中文名称显示）
- 板块选股功能（支持基于板块的选股，主要优势）
- 盘中辅助信息获取
- 实时快照获取（最快）
- 历史K线补充（非当天）
