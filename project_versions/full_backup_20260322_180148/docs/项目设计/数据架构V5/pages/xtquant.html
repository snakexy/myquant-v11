🚀 XtQuant - 当天分钟线专家

核心优势

  • 当天分钟线: 0.90ms 交易时间最优
  • 订阅功能: 300股 上限
  • L1快照: 6ms缓存读取

❌ 关键限制

  • 不支持板块数据 - 板块业务必须用TdxQuant/PyTdx2
  • 读取QMT本地缓存 - 非真正在线获取
  • 仅交易+盘后可用 - 周末/节假日无法使用

数据时效性说明

get_market_data_ex 实际是读取QMT本地缓存，并非真正"在线获取"。

  • 非交易时间：数据截止到最近一次收盘
  • 交易时间：取决于QMT是否自动更新当天数据

验证: 测试时间 2026-03-20 04:52（凌晨），最新数据：2026-03-19 15:00（昨天收盘）
