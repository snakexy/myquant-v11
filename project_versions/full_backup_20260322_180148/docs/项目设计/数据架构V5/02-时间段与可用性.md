## 2.1 时间段定义

### 完整时间划分

| 时间段类型 | 定义 | 具体范围 | 说明 |
|-----------|------|----------|------|
| **交易时间** | 周一至周五交易所开市 | 9:30-11:30, 13:00-15:00 | 所有数据源可用 |
| **盘后时间** | 交易日收盘后到晚上 | 15:00-24:00 | TdxQuant/XtQuant可用 |
| **非交易时间** | 周末、节假日、凌晨 | 00:00-9:00, 24:00-end | 仅PyTdx2/QLib可用 |

### 关键时间节点

| 节点 | 时间 | 说明 | 数据源影响 |
|------|------|------|-----------|
| **集合竞价** | 9:15-9:25 | 盘前竞价 | TdxQuant可用 |
| **开盘** | 9:30 | 正式开盘 | 所有数据源活跃 |
| **午间休息** | 11:30-13:00 | 午市休市 | TdxQuant保持连接 |
| **收盘** | 15:00 | 交易结束 | TdxQuant保持可用 |
| **盘后截止** | 约24:00 | 当日数据结束 | XtQuant可能断开 |
| **周末** | 周六、周日 | 市场休市 | 仅PyTdx2、QLib |

### 判断交易时间的代码

```python
from datetime import datetime, time
from enum import Enum

class TradingPhase(Enum):
    """交易阶段枚举"""
    PRE_MARKET = "盘前集合竞价"     # 9:15-9:25
    MORNING = "上午交易"            # 9:30-11:30
    LUNCH_BREAK = "午间休息"        # 11:30-13:00
    AFTERNOON = "下午交易"          # 13:00-15:00
    CLOSE = "收盘"                 # 15:00
    AFTER_HOURS = "盘后"           # 15:00-24:00
    NON_TRADING = "非交易时间"      # 其他时间

def get_trading_phase() -> TradingPhase:
    """获取当前交易阶段"""
    now = datetime.now()
    current_time = now.time()
    weekday = now.weekday()

    # 周末是非交易时间
    if weekday >= 5:  # 周六、周日
        return TradingPhase.NON_TRADING

    # 定义关键时间点
    pre_market_start = time(9, 15)
    morning_start = time(9, 30)
    morning_end = time(11, 30)
    afternoon_start = time(13, 0)
    afternoon_end = time(15, 0)
    after_hours_end = time(24, 0)

    # 判断交易阶段
    if pre_market_start <= current_time < morning_start:
        return TradingPhase.PRE_MARKET
    elif morning_start <= current_time < morning_end:
        return TradingPhase.MORNING
    elif morning_end <= current_time < afternoon_start:
        return TradingPhase.LUNCH_BREAK
    elif afternoon_start <= current_time < afternoon_end:
        return TradingPhase.AFTERNOON
    elif afternoon_end <= current_time < after_hours_end:
        # 15:00-24:00 收盘后也算交易相关的盘后时间
        return TradingPhase.AFTER_HOURS
    else:
        return TradingPhase.NON_TRADING

def is_trading_time() -> bool:
    """判断是否为可交易时间（TdxQuant/XtQuant可用）"""
    phase = get_trading_phase()
    return phase in [
        TradingPhase.PRE_MARKET,
        TradingPhase.MORNING,
        TradingPhase.LUNCH_BREAK,
        TradingPhase.AFTERNOON,
        TradingPhase.CLOSE,
        TradingPhase.AFTER_HOURS
    ]
```

## 2.2 数据源可用性矩阵

### 完整可用性表

| 数据源 | 交易时间 | 盘后时间 | 非交易时间 | 备注 |
|--------|----------|----------|-----------|------|
| **TdxQuant** | ✅ | ✅ | ❌ | 需要通达信终端运行 |
| **XtQuant** | ✅ | ✅ | ❌ | 需要QMT运行 |
| **PyTdx2** | ✅ | ✅ | ✅ | **24/7可用** |
| **QLib** | ✅ | ✅ | ✅ | 本地数据库 |

| 非交易时间 | ❌ | ❌ | ✅ 10-19ms | **PyTdx2** 🌟 |

| 非交易时间 | ❌ | ❌ | ✅ 10-19ms | **PyTdx2** 🌟 |

## 关键要点

### 1. TdxQuant的限制

- ❌ 非交易时间不可用
- ❌ 不支持当天分钟线
- ✅ 交易+盘后时间的快照和板块王者

### 2. XtQuant的限制

- ❌ 非交易时间不可用
- ❌ 不支持板块数据
- ✅ 当天分钟线最优

### 3. PyTdx2的价值

- ✅ **24/7可用**
- ✅ 非交易时间唯一的在线数据源
- ✅ 全功能支持

### 4. QLib的地位

- ✅ **24/7可用**
- ✅ 历史数据首选（7-10ms本地库）
