## 📝 备注

### 技术选型
- **后端**: Python + FastAPI
- **缓存**: Redis（推荐）/ Memory Dict
- **计算**: NumPy + Pandas + 并行计算
- **定时任务**: APScheduler

1. 大规模因子计算性能
2. 缓存一致性
3. 内存管理
