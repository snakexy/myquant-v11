## 模块定位

### 设计理念
在Research阶段，AI智能调参主要应用于因子权重优化、模型超参数调优和策略参数搜索，通过自动化搜索找到最优配置。

### 应用范围
```
因子计算模块 ────→ AI优化因子权重组合
    ↓
机器学习训练 ────→ AI超参数自动调优
    ↓
RL策略优化 ─────→ AI策略参数搜索
    ↓
因子评估 ───────→ AI评分权重优化
```

## 功能设计

### 1. 因子权重优化

**应用场景**: 因子计算模块
**优化目标**: 多因子组合的最优权重配置

| 输入 | 输出 | 优化方法 |
|-----|------|---------|
| 历史因子收益序列 | 最优因子权重 | 遗传算法/贝叶斯优化 |
| 因子IC值序列 | 权重调整建议 | 均值-方差优化 |
| 因子相关性矩阵 | 分散化权重 | 风险平价模型 |

**API设计**:
```
POST /api/v1/research/ai-tuning/factor-weights
{
  "factor_pool": ["momentum_5d", "volatility_20d", "turnover_rate"],
  "lookback_days": 252,
  "optimization_target": "sharpe_ratio"
}
```

### 2. 超参数调优

**应用场景**: 机器学习训练模块
**优化目标**: ML模型超参数配置

| 参数类型 | 搜索范围 | 优化方法 |
|---------|---------|---------|
| 学习率 | [1e-5, 1e-2] | 贝叶斯优化 |
| 网络层数 | [2, 8] | 网格搜索 |
| Dropout率 | [0.1, 0.5] | 随机搜索 |
| 批大小 | [32, 256] | 贝叶斯优化 |

**API设计**:
```
POST /api/v1/research/ai-tuning/hyperparams
{
  "model_type": "gru",
  "dataset_id": "csi300_5years",
  "search_space": {
    "learning_rate": {"min": 1e-5, "max": 1e-2, "type": "log"},
    "hidden_size": {"values": [64, 128, 256]},
    "num_layers": {"min": 2, "max": 4}
  },
  "n_trials": 50
}
```

### 3. RL策略参数优化

**应用场景**: RL策略优化模块
**优化目标**: 强化学习策略参数

| 参数类型 | 描述 | 优化方法 |
|---------|------|---------|
| 奖励函数权重 | 收益/风险/交易成本权重 | 进化算法 |
| 探索参数 | ε-greedy的ε值 | 贝叶斯优化 |
| 网络结构 | Actor/Critic网络配置 | 网格搜索 |

**API设计**:
```
POST /api/v1/research/ai-tuning/rl-params
{
  "strategy_id": "ppo_v2",
  "env_config": {...},
  "tuning_params": ["reward_weights", "exploration_rate"],
  "n_episodes": 1000
}
```

### 4. 因子评估权重优化

**应用场景**: 因子评估模块
**优化目标**: 评估指标权重配置

**API设计**:
```
POST /api/v1/research/ai-tuning/evaluation-weights
{
  "metrics": ["ic", "ir", "turnover", "drawdown"],
  "optimization_target": "factor_return",
  "constraints": {"turnover": {"max": 0.3}}
}
```

## 优化算法说明

### 1. 贝叶斯优化 (推荐)
- **适用场景**: 连续参数、计算开销大的情况
- **优势**: 样本效率高，收敛快
- **库**: Optuna, Hyperopt

### 2. 遗传算法
- **适用场景**: 离散/混合参数、多目标优化
- **优势**: 全局搜索能力强
- **库**: DEAP, PyGAD

### 3. 网格搜索
- **适用场景**: 参数空间小、需要全面搜索
- **优势**: 简单、可靠
- **库**: scikit-learn
