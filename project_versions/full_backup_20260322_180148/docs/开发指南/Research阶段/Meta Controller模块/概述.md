## 🎯 模块定位

### QLib官方定义
> **Meta Controller**提供指导给Forecast Model，旨在学习一系列预测任务之间的规律模式，并使用学到的模式指导即将到来的预测任务。

### 在三阶段中的位置
- **Research阶段**: 元学习框架系统，学习预测任务间规律模式
- **核心价值**: 适应市场动态变化，指导模型训练，提升预测性能

### 与其他模块的关系
- **依赖**: 因子评估模块（提供元信息）、模型训练模块（被指导）
- **协作**: AI助手模块（策略生成）
- **下游**: Validation阶段的Meta Model推理、Production阶段的在线Meta学习

### 在七层架构中的连接点
- **Layer 4（因子评估层）**: Meta Dataset从因子评估结果中获取元信息
- **Layer 6（模型训练层）**: Meta Model通过MetaGuideModel指导模型训练过程
- **Layer 8（初步验证层）**: 验证Meta Model指导效果

## 📊 核心功能

### QLib三大核心组件

#### 1. Meta Task ⭐
- 创建元学习任务实例
- 封装训练数据和元信息
- 支持多种模式（full/test/transfer）
- 使用`prepare_task_data()`获取Meta Model输入数据

#### 2. Meta Dataset ⭐
- 控制元信息生成过程
- 为Meta Model训练提供数据
- 使用`prepare_tasks()`获取Meta Task实例列表
- 支持跨数据集模式迁移

#### 3. Meta Model ⭐
- **MetaTaskModel**: 修改任务定义（动态调整训练任务）
- **MetaGuideModel**: 指导训练过程（在训练期间提供指导）
- 学习历史预测任务间的规律模式
- 使用学到的模式指导新的预测任务

### QLib官方示例：DDG-DA
- 适应市场动态变化的元学习算法
- 四步骤工作流：
  1. 计算元信息并封装到Meta Task
  2. 基于Meta Dataset训练DDG-DA
  3. DDG-DA推理获取指导信息
  4. 应用指导信息改善预测模型性能

### 输入输出

**输入**:
- 因子评估结果（IC、IR等元信息）
- 历史预测任务数据
- 模型训练配置

**输出**:
- Meta Task实例列表
- Meta Model训练结果
- Meta指导信息
- 优化后的预测模型

### 依赖哪些模块
- **因子评估模块** - 提供IC/IR指标作为元信息
- **因子分析模块** - 提供因子分析结果
- **模型训练模块** - 接收Meta指导信息

### 依赖哪些外部库
- **qlib** - Meta Controller核心框架
- **numpy/pandas** - 数据处理

### 被哪些模块依赖
- Validation阶段 - Meta Model推理验证
- Production阶段 - 在线Meta学习
