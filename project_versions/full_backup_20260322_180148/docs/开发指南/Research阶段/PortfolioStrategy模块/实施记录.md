**API 端点**:
- `GET /health` - 健康检查
- `GET /strategies` - 获取策略列表
- `GET /config` - 获取配置
- `POST /config` - 保存配置
- `POST /generate` - 生成投资组合
- `POST /backtest` - 策略回测
- `POST /optimize` - 参数优化
