# 因子评估模块概述

> **阶段**: Research阶段
> **最后更新**: 2026-02-11

---

## 🎯 模块定位

### 在三阶段中的位置
- **Research阶段**: 负责因子回测和性能评估
- **核心价值**: 评估因子策略的盈利能力和风险

### 与其他模块的关系
- **上游**: 因子分析模块 - 提供因子分析结果
- **下游**: AI助手模块 - 提供评估报告

---

## 📊 核心功能

### 功能列表

#### 1. 因子回测
- 单因子回测
- 多因子组合回测
- 分层回测
- 时间序列回测

#### 2. 性能评估
- 收益率计算
- 风险指标（夏普比率、最大回撤、波动率）
- 信息比率（IR）
- 基准比较

#### 3. 因子筛选
- IC值筛选
- IR值筛选
- 换手率筛选
- 综合评分

#### 4. 报告生成
- 回测报告
- 性能图表
- 因子排名

### 输入输出

**输入**:
- 因子数据
- 回测参数
- 基准数据

**输出**:
- 回测结果
- 性能指标
- 评估报告

---

### 依赖哪些模块
- **因子分析模块** - 提供因子分析结果

- **QLib** - 回测框架

### 被哪些模块依赖
