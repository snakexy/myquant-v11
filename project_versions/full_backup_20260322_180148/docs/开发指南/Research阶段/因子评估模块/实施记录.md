## 📝 技术亮点

### 1. 评分机制设计
- **多指标加权评分**: IC均值(40%) + IR(30%) + IC正数占比(30%)
- **阈值判断**: 通过/未通过清晰标记
- **建议生成**: 自动根据未通过的指标生成优化建议

### 2. 组合评估方法
- **等权重组合**: 简单平均
- **IC加权组合**: 按IC均值比例分配
- **自定义权重**: 灵活配置
- **权重归一化**: 自动确保和为1

### 3. 依赖注入设计
- 尝试加载data_service和factor_analysis_service
- 优雅降级：依赖不可用时使用模拟数据
- 保持服务可用性
