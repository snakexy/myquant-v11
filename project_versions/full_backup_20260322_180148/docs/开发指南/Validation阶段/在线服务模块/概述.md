## 🎯 模块定位

### 在三阶段中的位置
- **Validation阶段**: 提供在线滚动训练和信号生成能力
- **核心价值**: P1核心模块，连接Research和Production

### 与其他模块的关系
- **上游**: Research阶段机器学习训练模块 - 提供训练好的模型
- **下游**: Production阶段ML信号生成器模块 - 提供在线预测

---

## 📊 核心功能

### Online Serving组件体系 (基于QLib)

#### 1. OnlineManager（总控制器）
- first_train() - 首次训练
- routine() - 例行更新
- prepare_signals() - 准备信号
- get_signals() - 获取交易信号

#### 2. RollingStrategy（滚动策略）
- prepare_tasks() - 生成滚动任务
- get_next_rolling_time() - 下次滚动时间
- 滑动窗口训练逻辑

#### 3. OnlineLearningService（在线学习服务）
- start_online_learning() - 启动在线学习
- update_model() - 更新模型
- evaluate_performance() - 评估性能
- rollback_model() - 模型回滚

#### 4. SignalGenerator（信号生成器）
- generate_signals() - 生成交易信号
- predict() - 模型预测
- _calculate_confidence() - 计算置信度

#### 5. DataLoader（数据加载器）
- load() - 加载训练数据
- load_rolling_data() - 加载滚动数据

#### 6. QlibTrainerAdapter（训练适配器）
- train() - 训练模型
- predict() - 模型预测

### 输入输出

**输入**:
- 训练好的ML模型
- 历史行情数据
- 滚动策略配置

**输出**:
- 在线交易信号
- 更新后的模型
- 性能评估报告
