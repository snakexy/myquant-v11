## 🎯 模块定位

### 在三阶段中的位置
- **Validation阶段**: 实时监控策略运行状态
- **核心价值**: 提供性能监控和异常检测能力

### 与其他模块的关系
- **上游**: 在线服务模块、模拟实盘模块 - 监控目标
- **下游**: 预警系统模块 - 触发预警

---

## 📊 核心功能

### 功能列表

#### 1. 策略性能监控
- 实时收益率计算
- 回撤监控
- 夏普比率跟踪
- 胜率和盈亏比

#### 2. 异常检测
- 阈值检测（回撤、收益率）
- 3-sigma统计检测
- 异常记录管理

#### 3. 监控报告
- 实时指标报告
- 指标历史统计
- 异常汇总报告

#### 4. 策略状态管理
- 状态更新接口
- 资金和交易统计
- 峰值资金跟踪

### 核心类
- **MonitoringService**: 实时监控服务主类
- **MonitoringConfig**: 监控配置
- **MetricValue**: 指标值
- **AnomalyRecord**: 异常记录

### 输入输出

**输入**:
- 策略运行状态（资金、交易结果）
- 监控配置

**输出**:
- 实时指标
- 异常检测结果
- 监控报告
