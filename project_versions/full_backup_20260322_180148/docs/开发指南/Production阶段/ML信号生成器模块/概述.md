## 🎯 模块定位

### 核心价值
**将ML预测（0-1评分）转化为可执行的交易信号**

> **关键洞察**: ML模块不输出简单涨跌，而是输出**买入评分**，需要转换为交易信号

### 与QLib高级模块的集成

本模块深度集成QLib高级模块，实现生产级ML服务：

| QLib高级模块 | 集成方式 | 作用 |
|-------------|---------|------|
| **Workflow管理模块** | 定时触发预测流程 | 自动化日频/日内预测调度 |
| **Online Serving** | QLib内置在线服务 | 高性能实时预测（<50ms） |

### 与Research阶段的区别

| 方面 | Research阶段 | Production阶段 |
|------|-------------|---------------|
| **目标** | 训练最优模型 | 使用模型实时预测 |
| **数据** | 历史数据 | 实时/准实时数据 |
| **延迟要求** | 无（批量训练） | **<50ms**（实时预测） |
| **模型管理** | 单个模型 | **多版本管理**（v1.0/v1.1/v2.0） |
| **更新策略** | 重新训练 | **在线学习**、热加载 |
| **性能监控** | 离线评估 | **实时IC跟踪** |

### 在生产流程中的位置

```
因子计算 → 在线预测 → 信号生成 → 仓位管理 → 风险控制 → 实盘交易
    ↓           ↓           ↓           ↓           ↓          ↓
  技术指标   0-1评分    买入/卖出    动态仓位    风险检查    订单执行
```

## 📊 核心功能详解

### 1. Online Model Manager - 在线模型管理

**核心能力**:
- 模型版本管理（v1.0, v1.1, v2.0...）
- 模型热加载（不中断服务）
- 模型A/B测试
- 模型回滚

### 2. Online Prediction Service - 实时预测服务

**核心能力**:
- 实时单股票预测（延迟 < 50ms）
- 批量预测（4000只股票）
- 预测评分输出（0-1买入评分）

**性能要求**:
- 实时预测: < 50ms (P95)
- 批量预测: < 5s (4000只股票)
- 模型加载: < 100ms (热加载)

### 3. ML信号生成器

**核心能力**:
- 将ML预测评分（0-1）转换为交易信号
- 结合其他因子（技术指标、风险）生成综合信号
- 提供信号强度分级（WEAK/NORMAL/STRONG）
- 支持多种信号类型（买入、卖出、持有、观望）

### 4. ML驱动的仓位管理

**核心能力**:
- 根据ML预测置信度动态调整仓位
- 凯利公式变种
- 风险控制（单股票最大20%）

### 5. ML风险控制

**核心能力**:
- 评估ML预测组合的集中度风险
- 检测预测分布异常
- 基于预测置信度调整风险敞口

## 📊 信号分级

| 评分范围 | 信号类型 | 强度 | 建议仓位 | 目标价 |
|---------|---------|------|---------|--------|
| **0.8-1.0** | 强烈买入 | STRONG | 100% | +2% |
| **0.6-0.8** | 买入 | NORMAL | 100% | +1% |
| **0.4-0.6** | 观望/弱买入 | WEAK | 50% | +0.5% |
| **0.2-0.4** | 观望/弱卖出 | WEAK | 50% | -0.5% |
| **0.0-0.2** | 强烈卖出 | STRONG | 0% | -2% |

## 📊 监控指标

| 指标 | 说明 | 告警阈值 |
|------|------|----------|
| 实时IC | 每分钟计算 | IC < 0.02 或衰减60% |
| 预测延迟 | 每个预测测量 | 延迟 > 100ms |
| 预测准确率 | 实时验证 | 准确率 < 70% |
| 预测分布 | 检查分布偏移 | 偏移 > 20% |
