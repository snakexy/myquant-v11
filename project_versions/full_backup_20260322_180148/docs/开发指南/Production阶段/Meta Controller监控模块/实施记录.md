## 📋 实施概要

### 模块定位
实时监控ML模型性能，自动检测模型衰减，实现智能模型切换。

**核心价值**: ML模型上线后会逐渐衰减，需要持续监控和自动切换机制。

### 核心功能
1. **实时IC/IR跟踪** - 每分钟计算模型预测的IC和IR
2. **模型衰减检测** - IC下降、分布偏移、准确率下降检测
3. **自动模型切换** - 检测到严重衰减时自动切换备用模型
4. **告警管理** - 发送警告和严重告警通知

## ⚠️ 技术要点

### IC计算
- 使用Spearman秩相关系数
- 公式: `IC = corr(rank(预测评分), rank(实际收益))`

### 衰减检测阈值
| 指标 | 正常 | 警告 | 严重 |
|------|------|------|------|
| IC | > 0.03 | 0.02-0.03 | < 0.02 |
| IC衰减 | < 30% | 30-60% | > 60% |

### 自动切换策略
- IC < 0.02 或 衰减 > 60% → 自动切换
- IC < 0.03 或 衰减 > 40% → 发送警告
