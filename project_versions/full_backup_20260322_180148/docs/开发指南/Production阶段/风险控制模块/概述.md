## 🎯 模块定位

### 在三阶段中的位置
- **Production阶段**: 真实生产环境交易执行
- **核心价值**: 提供稳定可靠的实盘交易能力，集成qlib高级风险分析

### 与其他模块的关系
- **上游**: Validation阶段各模块 - 验证通过的策略
- **下游**: 无（最终执行层）

- **集成**: qlib_core/risk_management/risk_exposure.py - 因子暴露管理

---

## 📊 核心功能

### 功能列表

#### 1. 实盘交易
- 真实订单执行
- 券商接口对接
- 交易确认

#### 2. 基础风险控制
- 实时风险监控
- 仓位限制（总仓位/单一持仓）
- 止损止盈执行
- 集中度风险监控

#### 3. 高级风险分析（基于qlib）
- **VaR分析**: 历史模拟法/参数法/蒙特卡洛模拟
- **CVaR分析**: 条件风险价值（尾部风险）
- **Beta分析**: 组合相对沪深300的敏感度
- **因子暴露**: 规模/价值/动量/Beta因子监控
- **压力测试**: 多情景极端市场模拟

#### 4. 风险处置行动
- 预警通知（MONITOR/WARNING）
- 减仓操作（REDUCE_POSITION）
- 停止交易（STOP_TRADING）
- 强制平仓（FORCE_CLOSE）
- 紧急退出（EMERGENCY_EXIT）

#### 5. 性能监控
- 交易性能跟踪
- 系统状态监控
- 告警通知

### 输入输出

**输入**:
- 交易信号
- 风险参数
- 模型配置
- 历史收益率数据

**输出**:
- 订单执行结果
- 风险分析报告
- VaR/CVaR指标
- 压力测试结果
- 因子暴露报告

```
┌─────────────────────────────────────────────────────────────┐
│                    RiskService (Production)                  │

│  │ 实时风控 - 自动执行                                      ││

│  │ • 账户状态管理 → 交易权限控制                           ││
│  │ • 回调机制 → 外部系统集成                               ││

│                           ↕                                  │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ qlib RiskAnalyzer (高级分析)                            ││
│  │ • VaR/CVaR计算（历史/参数/蒙特卡洛）                    ││
│  │ • 压力测试（多情景模拟）                                ││
│  │ • Beta系数计算                                          ││
│  └─────────────────────────────────────────────────────────┘│
│                           ↕                                  │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ RiskExposureManager (因子暴露)                          ││
│  │ • 因子暴露计算                                          ││
│  │ • 暴露限制检查                                          ││
│  │ • 风险预算管理                                          ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

### 风险规则类型

| 风险类型 | 触发条件 | 默认行动 |
|---------|---------|---------|
| POSITION_LIMIT | 总仓位>95% | REDUCE_POSITION |
| POSITION_LIMIT | 单一持仓>20% | WARNING |
| LOSS_LIMIT | 总亏损<-15% | STOP_TRADING |
| DRAWDOWN_LIMIT | 回撤<-20% | STOP_TRADING |
| CONCENTRATION | 前5大持仓>50% | WARNING |
| VAR_LIMIT | VaR(95%)>5% | WARNING |
| CVAR_LIMIT | CVaR>8% | REDUCE_POSITION |
| BETA_EXPOSURE | Beta>1.5 | WARNING |
| FACTOR_EXPOSURE | 因子暴露>0.5 | WARNING |

## 🌐 API端点

### 基础风险控制API
| 方法 | 端点 | 说明 |
|-----|------|------|
| GET | /production/risk/rules | 列出风险规则 |
| GET | /production/risk/rules/{id} | 获取单个规则 |
| POST | /production/risk/rules | 添加规则 |
| DELETE | /production/risk/rules/{id} | 删除规则 |
| POST | /production/risk/monitoring/start | 启动监控 |
| POST | /production/risk/monitoring/stop | 停止监控 |
| GET | /production/risk/events | 获取风险事件 |
| GET | /production/risk/metrics | 获取风险指标 |

### 高级风险分析API
| 方法 | 端点 | 说明 |
|-----|------|------|
| GET | /production/risk/analysis/report/{account_id} | 完整风险分析报告 |
| GET | /production/risk/analysis/var/{account_id} | VaR分析 |
| GET | /production/risk/analysis/cvar/{account_id} | CVaR分析 |
| GET | /production/risk/analysis/beta/{account_id} | Beta分析 |
| GET | /production/risk/analysis/factor-exposure/{account_id} | 因子暴露分析 |
| POST | /production/risk/analysis/stress-test/{account_id} | 压力测试 |
| GET | /production/risk/analysis/overview/{account_id} | 风险概览(Dashboard) |
