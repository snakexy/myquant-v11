## 🧩 组件详细定义

### 1. RealtimeQuotes.vue — 看盘主页面

**组件路径**: `frontend/src/views/RealtimeQuotes.vue`

**Props**: 无（顶层页面组件）

**状态管理**:

| ref | 类型 | 说明 |
|-----|------|------|
| selectedStock | string | 当前选中股票代码 |
| currentTimeframe | string | 当前K线周期 |
| loading | boolean | 加载状态 |
| currentQuote | object | 当前行情快照 |
| marketPhase | string | 当前市场时段 |
| refreshInterval | number | 动态刷新间隔（秒） |
| watchlist | array | 自选股列表 |
| asks / bids | array | 五档盘口数据 |
| chart | ChartApi | lightweight-charts 实例 |

**生命周期**:

```
onMounted
  ├── fetchMarketStatus()       获取市场状态 → 设置刷新间隔
  ├── initChart()               创建图表实例
  ├── loadKlineData()           首次加载K线
  └── startUpdateTimer()        按市场状态动态轮询

onBeforeUnmount
  └── stopUpdateTimer()         清理轮询定时器
```

**核心方法**:

| 方法 | 说明 | 调用API |
|------|------|---------|
| fetchMarketStatus() | 获取市场状态 | `GET /api/v1/quotes/market/status` |
| loadKlineData() | 加载K线数据 + 快照 | `GET /api/v1/quotes/kline/{symbol}` + `GET /api/v1/quotes/snapshot/{symbol}` |
| updateOrderBookFromSnapshot() | 从快照更新五档盘口 | 内部处理 |
| changeTimeframe(period) | 切换K线周期 | 重新调用 loadKlineData() |
| selectStock(code) | 切换选中股票 | 重新调用 loadKlineData() |
| addStock() | 添加股票到自选 | 仅本地操作 |
| formatVolume(vol) | 格式化成交量 | 万/亿 |
| formatAmount(amt) | 格式化成交额 | 万/亿 |

## 📡 API 调用规范

### 市场状态驱动的动态刷新

前端应根据市场状态 API 动态调整轮询频率，而不是固定 5 秒：

```typescript
// 获取市场状态，动态设置刷新间隔
const marketStatus = await fetchMarketStatus()
// 交易时间: 3-5秒刷新
// 午休/收盘后: 30秒刷新
// 周末: 30秒刷新
updatePollingInterval(marketStatus.refresh_interval)
```

**UI 显示**:
- 状态栏显示当前市场状态（交易中 / 午休 / 休市）
- 集合竞价时段特殊标识
- 非交易时间显示"非交易时段"提示

### 当前问题

```typescript
// ❌ 硬编码 API 路径
const API_BASE = '/api/v1/quotes'
const res = await fetch(`${API_BASE}/kline/${symbol}?period=${period}&count=800`)
```

### 正确方式

```typescript
// ✅ 通过 api/modules/ 统一调用
import { fetchKline, fetchSnapshot } from '@/api/modules/quotes'

const klineData = await fetchKline(symbol, period, count)
const snapshotData = await fetchSnapshot(symbol)
```

**待创建文件**: `frontend/src/api/modules/quotes.ts`

```typescript
// frontend/src/api/modules/quotes.ts
import http from '@/api/http'

export function fetchKline(symbol: string, period: string = '1d', count: number = 800) {
  return http.get(`/api/v1/quotes/kline/${symbol}`, { params: { period, count } })
}

export function fetchSnapshot(symbol: string) {
  return http.get(`/api/v1/quotes/snapshot/${symbol}`)
}

export function fetchSnapshotBatch(symbols: string[]) {
  return http.get('/api/v1/quotes/snapshot', { params: { symbols: symbols.join(',') } })
}
```

## 📝 自选股管理

### 当前问题

```typescript
// ❌ 自选股硬编码
const watchlist = ref([
  { code: '600000.SH', name: '浦发银行', price: '--', change: 0 },
  { code: '000001.SZ', name: '平安银行', price: '--', change: 0 },
  // ...
])
```

### 正确方式

```typescript
// ✅ localStorage 持久化
import { ref, watch } from 'vue'

const WATCHLIST_KEY = 'myquant_watchlist'

const defaultWatchlist = [
  { code: '600000.SH', name: '浦发银行', price: '--', change: 0 },
  { code: '000001.SZ', name: '平安银行', price: '--', change: 0 },
]

// 从 localStorage 读取
const watchlist = ref(JSON.parse(localStorage.getItem(WATCHLIST_KEY)) || defaultWatchlist)

// 变化时自动保存
watch(watchlist, (val) => {
  localStorage.setItem(WATCHLIST_KEY, JSON.stringify(val))
}, { deep: true })
```
