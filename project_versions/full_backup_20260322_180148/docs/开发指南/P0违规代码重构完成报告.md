### 2. hotspot_88sectors_with_stocks.py (v1.0 → v2.0)

**违规位置**:
- Line 38: `from xtquant import xtdata`
- Line 100: `stocks = xtdata.get_stock_list_in_sector(sector_name)`

**重构方案**:
```python
# ❌ 旧代码 (v1.0)
from xtquant import xtdata

def get_sector_stocks_with_xtquant(code: str, fallback_stocks: list) -> list:
    if XTQUANT_AVAILABLE:
        stocks = xtdata.get_stock_list_in_sector(sector_name)
        if stocks and len(stocks) > 0:
            return stocks
    return fallback_stocks

# ✅ 新代码 (v2.0)
from data.unified_data_manager import get_unified_data_manager, DataType

def get_sector_stocks_unified(code: str, fallback_stocks: list) -> list:
    manager = get_manager()
    result = manager.get_data(
        data_type=DataType.SECTOR_COMPONENTS,
        symbols=[sector_name],
        context={'limit': 100}
    )
    if result and sector_name in result:
        stocks = result[sector_name]
        if stocks and len(stocks) > 0:
            return stocks
    return fallback_stocks
```

**架构合规性**:
- ✅ 使用UnifiedDataManager获取板块成分股
