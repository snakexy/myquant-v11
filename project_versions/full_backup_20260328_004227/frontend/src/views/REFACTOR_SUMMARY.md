## 组件功能说明

### 1. OverviewCards
**功能:** 显示数据概览
- 总数据量（显示数据库详情：通达信、Qlib）
- 数据完整度
- 活跃数据源（显示数据源列表）
- 今日更新

**数据来源:** `/api/v1/data-management/database/stats`

### 2. StockCategories
**功能:** 股票分类统计
- 沪深300、中证500、上证50、创业板指、科创50
- 每个分类显示：股票数量、平均涨幅、热门股票、性能图表

**数据来源:** `/api/v1/data-management/categories/stats`

**子组件:** StockCategoryCard

### 3. DataQualityMonitor
**功能:** 数据质量监控
- 监控6类数据质量：日线、分钟线、分笔、财务、指数、板块
- 每类显示：完整度、准确率、及时性、状态指示

**特性:** 自动检测质量状态（正常/警告/异常）

### 4. DataSourceManagement
**功能:** 数据源管理
- 显示所有数据源（智能缓存、通达信本地/远程、Qlib等）
- 操作：启用/禁用、测试连接、配置
- 显示：更新频率、错误数、最后更新时间

**数据来源:** `/api/v1/data-management/sources/list`
