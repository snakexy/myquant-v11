# 待实现：双击股票数据管理弹窗

## 需求描述

旧版中双击自选股列表的股票，会弹出一个数据管理窗口，功能包括：
- **删除指定周期数据**：删除 HotDB 中某只股票某个周期的错误数据
- **重新下载数据**：从在线源重新下载正确的数据
- **数据修复**：用于修复 HotDB 中存储的错误数据

## 使用场景

当发现某只股票的某个周期数据有问题时（如缺口、错误值等），可以通过此弹窗：
1. 选择该股票
2. 选择有问题的周期（1m、5m、1d等）
3. 删除错误数据
4. 重新从在线源下载正确数据
5. 数据会保存到 HotDB
