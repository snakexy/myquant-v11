## 4. 板块成分股功能

### get_stock_list_in_sector() - 获取板块成分股

**功能描述**: 获取指定板块的成分股列表

**优势**:
- ✅ **板块选股**: 支持基于板块的选股
- ✅ **实时更新**: 成分股实时更新
- ✅ **准确可靠**: 来自通达信官方数据

**获取方式**:
```python
# 获取板块成分股
bank_stocks = tdxquant.get_stock_list_in_sector("银行")

# 返回结果示例：
# ['000001.SZ', '002142.SZ', '601166.SH', '601398.SH', ...]
# 共42只银行股
```

**板块选股应用**:
```python
# 基于板块的多因子选股
def sector_factor_screen(sector_name, filters):
    """基于板块的多因子选股"""
    # 获取板块成分股
    stocks = tdxquant.get_stock_list_in_sector(sector_name)

    # 获取每只股票的辅助信息
    selected = []
    for stock in stocks:
        info = tdxquant.get_more_info(stock)
        # 应用筛选条件
        if filters['PE'] and info['DynaPE'] < filters['PE']:
            selected.append(info)

    return selected

# 使用示例
value_stocks = sector_factor_screen("银行", {'PE': 10})
```

**与XtQuant对比**:
- **TdxQuant**: ✅ 完整支持板块选股 ✅
- **XtQuant**: ❌ 无此功能

### 3. 板块轮动分析

**场景**: 需要分析板块轮动和热点切换

**推荐数据源**: **TdxQuant**（完整板块数据）

**使用方式**:
```python
# 获取所有板块
all_sectors = tdxquant.get_sector_list()

# 获取各板块实时行情
sector_performance = []
for sector_code in all_sectors:
    name = get_sector_name_from_code(sector_code)
    quote = tdxquant.get_sector_quote(sector_code)
    sector_performance.append({
        'code': sector_code,
        'name': name,
        'change_percent': quote['change_percent']
    })

# 排序识别热点板块
sorted_sectors = sorted(sector_performance,
                       key=lambda x: x['change_percent'],
                       reverse=True)
hot_sector = sorted_sectors[0]
print(f"热点板块: {hot_sector['name']}, 涨跌幅: {hot_sector['change_percent']}%")
```
