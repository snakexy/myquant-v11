## 板块数据功能

### 1. 板块指数数据

**功能描述**: 获取各类板块指数的实时和历史行情

**获取方式**:
```python
# 获取板块指数实时行情
sector_index_data = tdxquant.get_sector_index_data('申万一级')

# 获取板块指数历史行情
sector_index_history = tdxquant.get_sector_index_history('AI概念', period='1d', count=100)
```

**数据内容**:
- 板块指数代码
- 板块指数名称
- 当前价格
- 涨跌幅
- 成交量
- 成交额
- 最高价
- 最低价
- 开盘价
- 收盘价

**性能**:
- 实时快照: 0.60ms（最快）
- 历史K线: 35-50ms（1088根）

### 2. 板块成分股列表

**功能描述**: 获取各板块的成分股列表

**获取方式**:
```python
# 获取板块成分股列表
component_stocks = tdxquant.get_sector_components('申万一级', '银行')

# 返回结果
# ['600000.SH', '601166.SH', '601398.SH', ...]
```

**数据内容**:
- 成分股代码
- 成分股名称
- 在板块中的权重（如果提供）

**性能**:
- 成分股查询: 待测试
- 预计: 10-50ms

### 3. 板块实时行情

**功能描述**: 获取板块指数的实时快照

**获取方式**:
```python

```

**数据内容**:
- 实时价格
- 涨跌幅

- 买卖盘口（五档）

**性能**:
- 实时快照: 0.60ms（最快）
