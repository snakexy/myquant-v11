## 核心需求 vs 官方API映射

### 需求1: 获取实时行情（L1）

#### 官方方案A: subscribe_quote + get_full_tick ✅

**文档位置**: 实时行情相关

**官方示例**:
```python
# 1. 订阅
from xtquant import xtdata
xtdata.subscribe_quote('600000.SH', period='1d', count=0)

# 2. 获取
data = xtdata.get_full_tick(['600000.SH'])
```

**官方说明**:
- `subscribe_quote()`: 订阅指定股票的行情
- `get_full_tick()`: 获取订阅股票的实时行情
- 适合: 高频访问场景

#### 官方方案B: get_market_data

**文档位置**: 市场数据获取

**官方示例**:
```python
data = xtdata.get_market_data(
    field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
    stock_list=['600000.SH'],
    period='1d',
    start_time='',
    end_time='',
    fill_data=True
)
```

**官方说明**:
- 获取市场数据（包括实时）
- 需要先用 subscribe_quote 订阅

### 需求2: 获取历史K线（L2/L3）

#### 官方方案: get_market_data_ex ✅

**文档位置**: 历史数据获取

**官方示例**:
```python
# 获取历史K线
data = xtdata.get_market_data_ex(
    field_list=['open', 'high', 'low', 'close', 'volume', 'amount'],
    stock_list=['600000.SH'],
    period='1d',
    start_time=20200101,
    end_time=20201231,
    dividend_type='front'
)
```

**官方说明**:
- `start_time=0, end_time=0`: 获取本地所有数据
- `count=N`: 获取最新N条
- `dividend_type`: 'none'(不复权), 'front'(前复权), 'back'(后复权)

**注意事项**:
- 分钟线数据有限制（需要验证）
- 建议先下载历史数据

### 需求5: 订阅推送回调

#### 官方方案: register_callback ✅

**文档位置**: 订阅推送

**官方示例**:
```python
# 定义回调函数
def on_data(data):
    print("收到推送:", data)

# 注册回调
from xtquant import xtdata
xtdata.register_callback(on_data)

# 订阅
xtdata.subscribe_quote('600000.SH', period='tick')
```

**官方说明**:
- 回调函数会在数据更新时自动调用
- 支持tick级推送
- 适合实时监控场景

## 官方限制和注意事项

### 1. 订阅限制

**官方说明**:
> 单个MiniQMT实例订阅数量有限制
> 建议使用多实例模式扩展订阅能力

**应对方案**:
- 使用多实例并发订阅
- 智能管理订阅列表

### 2. 分钟线限制

**官方说明**:
> 分钟线数据在线获取限制为16天
> 超过16天需要先下载历史数据

**应对方案**:
- 自动下载历史数据
- 本地缓存优先

### 3. 数据更新频率

**官方说明**:
> L1实时数据: 约3秒更新一次
> tick数据: 实时推送

**应对方案**:
- 根据场景选择合适的更新频率

### 4. 时间参数格式

**官方说明**:
```python
# start_time/end_time 支持多种格式
start_time=20200101        # 整数: YYYYMMDD
start_time='20200101'      # 字符串: 'YYYYMMDD'
start_time=0               # 0: 不限制
```

## 官方推荐的使用模式

### 模式1: 首次使用流程

```python
from xtquant import xtdata

# 1. 连接
xtdata.connect()

# 2. 下载历史数据
xtdata.download_history_data(
    stock_code='600000.SH',
    period='1d',
    start_time='20200101',
    end_time='20241231'
)

# 3. 读取本地数据
data = xtdata.get_market_data_ex(
    stock_list=['600000.SH'],
    period='1d',
    count=100
)
```

### 模式2: 实时监控流程

```python
# 1. 定义回调
def on_quote_update(data):
    handle_update(data)

# 2. 注册回调
xtdata.register_callback(on_quote_update)

# 3. 订阅
xtdata.subscribe_quote('600000.SH', period='1d')

# 4. 持续运行
while True:
    time.sleep(1)
```

### 模式3: 混合模式（历史+实时）

```python
# 1. 先获取历史数据
history = xtdata.get_market_data_ex(
    stock_list=['600000.SH'],
    period='1d',
    count=100
)

# 2. 订阅实时数据
xtdata.subscribe_quote('600000.SH', period='1d')

# 3. 启动后台更新
def update_realtime():
    while True:
        latest = xtdata.get_full_kline(['600000.SH'], count=1)
        update_chart(latest)
        time.sleep(3)
```
