## 正确使用方式

### 场景1: 获取最近几天（实时）

```python
def get_recent_minute_kline(symbol, period='5m', count=1000):
    """
    获取最近几天分钟K线（实时数据）

    限制：最多约22.6个交易日（1083条5分钟K线）

    适用：实时行情、最近几天的数据
    """
    from xtquant import xtdata

    # 先订阅
    xtdata.subscribe_quote(symbol, period=period, count=0)

    # 再获取
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time='',
        end_time='',
        count=count,
        dividend_type='none',
        fill_data=True
    )

    return data.get(symbol)
```

### 场景2: 获取历史数据（下载+读取）⭐

```python
def get_historical_minute_kline(symbol, start_date, end_date, period='5m'):
    """
    获取历史分钟K线（下载+读取）

    无时间限制！可下载任意时间范围

    Args:
        symbol: 股票代码 '600519.SH'
        start_date: 开始日期 '20200101'
        end_date: 结束日期 '20241231'
        period: 周期 '1m', '5m', '15m', '30m', '60m'

    Returns:
        DataFrame

    示例:
        # 获取2024全年数据
        df = get_historical_minute_kline('600519.SH', '20240101', '20241231', '5m')
        # 11616条数据，40.33天
    """
    from xtquant import xtdata

    # 1. 下载历史数据
    xtdata.download_history_data(
        stock_code=symbol,
        period=period,
        start_time=start_date,  # 字符串格式 '20200101'
        end_time=end_date       # 字符串格式 '20241231'
    )

    # 2. 读取数据
    data = xtdata.get_market_data_ex(
        field_list=['time', 'open', 'high', 'low', 'close', 'volume', 'amount'],
        stock_list=[symbol],
        period=period,
        start_time=start_date,  # 字符串格式
        end_time=end_date,      # 字符串格式
        count=0,
        dividend_type='none'
    )

    return data.get(symbol)
```

### 场景3: 获取完整数据（混合策略）

```python
def get_complete_minute_kline(symbol, start_date, end_date, period='5m'):
    """
    获取完整分钟K线（历史+实时）

    策略：
    1. 下载历史数据（任意时间范围）
    2. 获取实时数据（最近22.6个交易日）
    3. 合并并去重

    Args:
        symbol: 股票代码
        start_date: 开始日期
        end_date: 结束日期
        period: 周期

    Returns:
        完整的DataFrame
    """
    import pandas as pd
    from datetime import datetime, timedelta

    # 1. 下载历史数据（start_date 到 昨天）
    yesterday = (datetime.now() - timedelta(days=1)).strftime('%Y%m%d')

    try:
        historical = get_historical_minute_kline(
            symbol, start_date, yesterday, period
        )
    except:
        historical = None

    # 2. 获取实时数据（最近几天）
    try:
        realtime = get_recent_minute_kline(symbol, period, count=1000)
    except:
        realtime = None

    # 3. 合并
    if historical is not None and realtime is not None:
        # 去重（实时数据会重复最后几天）
        complete = pd.concat([historical, realtime]).drop_duplicates()
        return complete.sort_index()
    elif historical is not None:
        return historical
    elif realtime is not None:
        return realtime
    else:
        return None
```

## 之前测试的误区

### ❌ 错误结论1: "16天限制"

**错误原因**: 混淆了在线获取和下载的限制

**正确理解**:
- 在线获取（count方式）：限制22.6个交易日
- 下载+读取：**无时间限制**

---

### ❌ 错误结论2: "下载只能获取3.67天"

**错误原因**: 时间范围选择不当

**错误做法**:
```python
# ❌ 用最近30天（包含很多非交易时间）
end_date = datetime.now()
start_date = end_date - timedelta(days=30)

xtdata.download_history_data(
    stock_code=symbol,
    period='5m',
    start_time=start_date.strftime('%Y%m%d'),
    end_time=end_date.strftime('%Y%m%d')
)
# 结果：只有3.67天数据
```

**正确做法**:
```python
# ✅ 用具体日期（确保覆盖交易日）
xtdata.download_history_data(
    stock_code=symbol,
    period='5m',
    start_time='20240101',  # 具体日期
    end_time='20241231'     # 具体日期
)
# 结果：11616条，40.33天
```
