## 🎯 模块定位

### 在三阶段中的位置
- **Validation阶段**: 提供虚拟账户交易模拟
- **核心价值**: 模拟实盘交易环境，与实盘仅差虚拟账户

### 与其他模块的关系
- **上游**: 在线服务模块 - 获取交易信号
- **下游**: Production阶段实盘交易模块 - 提供预演环境

---

## 📊 核心功能

### 功能列表

#### 1. 虚拟账户管理
- SimulationAccount - 模拟账户
- 资金管理（初始资金、可用资金、冻结资金）
- 市值计算

#### 2. 订单交易
- SimulationOrder - 模拟订单
- 订单类型：市价单、限价单、止损单
- 订单验证与执行

#### 3. 持仓管理
- SimulationPosition - 模拟持仓
- 实时价格更新
- 盈亏计算

#### 4. 统计分析
- 账户统计（收益率、胜率等）
- 交易记录查询
- 账户重置

### 核心类
- **SimulationService**: 模拟实盘服务主类
- **SimulationAccount**: 模拟账户
- **SimulationPosition**: 模拟持仓
- **SimulationOrder**: 模拟订单
- **SimulationConfig**: 配置

### 输入输出

**输入**:
- 交易信号（买卖指令）
- 行情价格
- 账户配置

**输出**:
- 订单执行结果
- 持仓状态
- 账户统计
