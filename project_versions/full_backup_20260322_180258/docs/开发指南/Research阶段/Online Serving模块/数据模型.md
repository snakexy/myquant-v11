## 3. CalculateTask - 计算任务

**描述**: 因子计算任务

```python
@dataclass
class CalculateTask:
    """计算任务"""
    task_id: str                        # 任务ID
    status: TaskStatus                  # 任务状态
    factor_names: List[str]             # 因子名称列表
    stock_codes: List[str]              # 股票代码列表
    start_date: str                     # 开始日期
    end_date: str                       # 结束日期
    use_cache: bool = True              # 是否使用缓存
    parallel: bool = True               # 是否并行计算
    created_at: datetime = None         # 创建时间
    started_at: datetime = None         # 开始时间
    completed_at: datetime = None       # 完成时间
    result: CalculateResult = None      # 计算结果

class TaskStatus(Enum):
    """任务状态"""
    PENDING = "pending"
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"
    CANCELLED = "cancelled"
```

```typescript
interface CalculateTask {
  taskId: string;
  status: 'pending' | 'running' | 'completed' | 'failed' | 'cancelled';
  factorNames: string[];
  stockCodes: string[];
  startDate: string;
  endDate: string;
  useCache: boolean;
  parallel: boolean;
  createdAt?: string;
  startedAt?: string;
  completedAt?: string;
  result?: CalculateResult;
}
```

## 📐 缓存键设计

### 键命名规则

```
{prefix}{factor_name}:{start_date}:{end_date}

示例:
factor:alpha360_001:2025-01-01:2025-12-31
factor:custom_factor_01:2025-06-01:2025-06-30
```

### 键结构

```python
def build_cache_key(factor_name: str, start_date: str, end_date: str) -> str:
    """构建缓存键"""
    return f"factor:{factor_name}:{start_date}:{end_date}"

def parse_cache_key(key: str) -> Tuple[str, str, str]:
    """解析缓存键"""
    parts = key.split(":")
    return parts[1], parts[2], parts[3]
```
