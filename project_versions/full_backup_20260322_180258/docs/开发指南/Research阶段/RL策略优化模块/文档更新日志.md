## 📝 更新规范

### 什么时候更新文档？

**必须更新**（强制）:
1. **完成功能模块改造后**
   - 例如: StrategyOptimizer真实数据回测改造
   - 必须: 更新实施记录.md
   - 必须: 更新概述.md（完成度）
   - 可选: 更新测试报告.md（如有测试）

2. **完成功能测试验证后**
   - 必须: 更新测试报告.md
   - 必须: 更新概述.md（如影响状态）

3. **决定重大方案变更后**
   - 必须: 更新真实数据实现方案.md
   - 必须: 更新概述.md
   - 必须: 更新实施记录.md

**不需要更新**:
- ❌ 代码微调（不改变功能）
- ❌ 格式调整（不影响内容）
- ❌ 临时文件创建/删除

### 更新流程

```
功能改造完成
    ↓
更新实施记录.md (详细记录)
更新概述.md (完成度)
    ↓
可选: 更新测试报告.md
```
