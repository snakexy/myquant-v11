## 🔧 实现细节

### Service层: `rl_strategy_service.py`

**核心类**:
- `RLStrategyService` - 主服务类

**数据模型**:
- `RLTrainingConfig` - 训练配置
- `RLTrainingResult` - 训练结果
- `RLOptimizationResult` - 优化结果
- `RLTrainingTrial` - 训练试验记录

**核心方法**:

1. **`train_rl_strategy()`** - RL策略训练
   - 支持DQN、PPO、A2C三种算法
   - 支持order_execution和portfolio_construction两种场景
   - 基于Tianshou框架实现
   - 提供模拟训练作为降级方案
   - 保存训练结果为.pkl格式

2. **`optimize_strategy()`** - 策略超参数优化
   - 网格搜索参数优化
   - 支持自定义参数网格
   - 自动选择最佳参数配置
   - 保存优化结果

3. **模型管理方法**:
   - `save_model()` - 保存模型
   - `load_model()` - 加载模型
   - `list_models()` - 列出所有模型

4. **训练历史管理**:
   - `get_training_result()` - 获取训练结果
   - `list_training_history()` - 列出训练历史
   - `get_statistics()` - 获取统计信息

**技术栈**:
- Tianshou（RL策略框架）
- PyTorch（深度学习）
- Gym（环境接口）
- NumPy（数值计算）

**存储路径**:

## 🔧 QLib RL框架详情

### 核心组件（基于QLib官方文档）

**EnvWrapper（环境封装）**:
- Simulator: 市场模拟器
  - SingleAssetOrderExecution（详细，慢）
  - SimpleSingleAssetOrderExecution（简化，快）
- State Interpreter: 状态转换器
- Action Interpreter: 动作转换器
- Reward Function: 奖励函数

**Policy（策略）**:
- 基于Tianshou策略框架
- 支持DQN、PPO、A2C等算法

**Training Vessel & Trainer（训练）**:
- Training Vessel: 包含所有组件
- Trainer: Scikit-learn风格接口（trainer.fit()）

### 应用场景

**1. Order Execution（订单执行优化）**:
- 单资产订单执行
- 多资产订单执行
- 状态: 订单簿、价格、成交量、波动率
- 动作: 订单大小、价格、时机
- 奖励: 价格优势、交易成本、市场影响、完成率

**2. Portfolio Construction（投资组合构建）**:
- 股票市场、加密货币、外汇
- 状态: 市场和投资组合信息
- 动作: 资本配置决策
- 奖励: 总回报、风险调整回报、夏普比率、回撤
