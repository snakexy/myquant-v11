# 因子分析模块概述

> **阶段**: Research阶段
> **最后更新**: 2026-02-13

---

## 🎯 模块定位

### 在三阶段中的位置
- **Research阶段**: 负责因子统计分析和相关性分析
- **核心价值**: 评估因子质量和预测能力

### 与其他模块的关系
- **上游**: 因子计算模块 - 提供因子数据
- **下游**:
  - 因子评估模块 - 提供分析结果
  - AI助手模块 - 提供因子分析报告

---

## 📊 核心功能

### 功能列表

#### 1. 描述性统计分析
- 均值、标准差、最大值、最小值
- 分位数分析（25%, 50%, 75%）
- 偏度和峰度
- 分布可视化

#### 2. 因子相关性分析
- 因子间相关性矩阵
- 热力图可视化
- 高相关因子识别

#### 3. 因子有效性分析
- IC（信息系数）计算
- IR（信息比率）计算
- 因子单调性测试
- 因子稳定性分析

#### 4. 因子分组分析
- 按行业分组
- 按市值分组
- 分层回测分析

### 输入输出

**输入**:
- 因子数据（从因子计算模块）
- 分析参数
- 时间范围

**输出**:
- 统计分析报告
- 相关性矩阵
- 有效性指标
