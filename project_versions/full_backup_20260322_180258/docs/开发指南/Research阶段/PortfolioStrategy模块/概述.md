## 📊 QLib Portfolio Strategy 简介

### 核心概念

Portfolio Strategy 是 **QLib 的核心组件之一**，专门用于根据预测分数生成投资组合。

**工作原理**:
```python
# QLib 官方示例
from qlib.contrib.strategy import TopkDropoutStrategy

strategy = TopkDropoutStrategy(
    topk=50,        # 持有50只股票
    n_drop=5,       # 每天卖出5只，买入5只
    signal=pred_score  # ML模型的预测分数
)
```

### QLib 内置策略

| 策略类 | 说明 | 参数 |
|--------|------|------|
| **TopkDropoutStrategy** | 选Top-K个股票，每天掉仓N个 | topk, n_drop, signal |
| **EnhancedIndexingStrategy** | 增强指数化，跑赢基准 | risk_thld, turn_rate |
| **WeightStrategyBase** | 权重策略基类 | - |

### Topk-Drop 算法说明

```
算法流程:
1. 根据预测分数对所有股票排名
2. 选择前 topk 名股票持有
3. 每个交易日:
   - 找出当前持仓中排名 > K 的股票（最差的 d 个）
   - 卖出这 d 个股票
   - 买入未持仓中排名最好的 d 个股票
   - 通常 d = n_drop
```

## 📊 核心功能

### 功能列表

#### 1. 策略配置管理 ⭐
- 配置策略类型（TopkDropout / EnhancedIndexing）
- 配置策略参数（topk, n_drop 等）
- 策略参数保存/加载

#### 2. 投资组合生成 ⭐
- 根据 ML 预测分数生成股票列表
- 计算目标持仓权重
- 生成交易订单列表

#### 3. 策略回测
- 在历史数据上测试策略
- 计算策略性能指标
- 生成回测报告

#### 4. 参数优化接口
- 提供参数优化接口（供 RL 使用）
- 返回策略性能指标
- 支持参数搜索

### 输入输出

**输入**:
- ML 预测分数 (prediction scores)
- 策略类型选择
- 策略参数配置

**输出**:
- 投资组合（股票列表 + 权重）
- 交易订单列表
- 策略性能指标
